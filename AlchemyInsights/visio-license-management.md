# <a name="visio-license-management"></a>Gerenciamento de licenças do Visio

O gerenciamento de licença do Visio pode ser concluído no **Centro de Administração do Microsoft 365**.

- Para **mudar de plano**, confira [Mudar de plano do Microsoft 365 for Business manualmente](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-plans-manually?view=o365-worldwide).
- Para **comprar** o Visio, vá para **Cobrança > [Comprar serviços](https://go.microsoft.com/fwlink/p/?linkid=868433)**. Use a caixa de Pesquisa para pesquisar "Visio"; selecione o plano e o número de usuários. Para obter mais informações, confira [Comprar assinatura](https://docs.microsoft.com/microsoft-365/commerce/buy-another-subscription?view=o365-worldwide).
- Para **atribuir** licenças do Visio, vá para **Cobrança > [Licenças](https://go.microsoft.com/fwlink/p/?linkid=842264)**. Para obter mais informações, confira [Atribuir licenças](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide).
- Para **adicionar/remover** licenças do Visio, vá para **Cobrança > [Adquirir serviços](https://go.microsoft.com/fwlink/p/?linkid=868433)**. Para obter mais informações, confira [Adicionar/remover licenças](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription).
- Para **cancelar** licenças do Visio, vá para **Cobrança > [Produtos e serviços](https://go.microsoft.com/fwlink/p/?linkid=842054)**, selecione **Configurações e ações** e clique em **Cancelar assinatura**. Para obter mais informações, confira [Cancelar sua assinatura](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription).
- [Compare as opções do Visio](https://products.office.com/Visio/microsoft-visio-plans-and-pricing-compare-visio-options) para saber mais sobre as diferenças entre o plano 1 e o plano 2 do Visio.

Para mais informações, confira [Gerenciar licenças de assinatura](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription).
