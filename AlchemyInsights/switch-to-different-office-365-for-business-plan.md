# <a name="switch-to-a-different-office-365-for-business-plan"></a>Migrar para um outro plano do Office 365 para empresas

A maneira mais fácil de mudar de plano é usar o botão **mudar de planos** no centro de administração.

- Use o botão [alternar planos](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-to-a-different-plan?view=o365-worldwide#use-the-switch-plans-button) para conduzir você por meio de um processo de mudança de planos. 

- Se o botão mudar de plano não estiver disponível, você pode tentar [mudar de plano manualmente](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-to-a-different-plan?view=o365-worldwide#the-switch-plans-button-isnt-there). 

Para mais informações, confira [Mudar para outro plano do Office 365 para empresas](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-to-a-different-plan?view=o365-worldwide).
