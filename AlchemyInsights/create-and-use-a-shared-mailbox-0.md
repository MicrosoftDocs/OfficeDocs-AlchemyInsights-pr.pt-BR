# <a name="create-and-use-a-shared-mailbox"></a>Criar e usar uma caixa de correio compartilhada

Uma caixa de correio compartilhada permite que mais de uma pessoa envie e receba emails para um alias, como o **Sales@contoso.com**, sem atribuir outra licença.
  
[Criar uma caixa de correio compartilhada](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) explica como: 
  
- Criar uma caixa de correio compartilhada
    
- Conceder aos usuários permissões para usar uma caixa de correio compartilhada
    
- Acessar a caixa de correio compartilhada no Outlook ou no Outlook na Web
