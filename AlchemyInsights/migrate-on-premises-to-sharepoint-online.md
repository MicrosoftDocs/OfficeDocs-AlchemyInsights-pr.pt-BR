# <a name="migrate-on-premises-to-sharepoint-online"></a>Migrar no local para o SharePoint Online

Para migrar de um farm do SharePoint local para o SharePoint Online, você pode usar a ferramenta de migração do SharePoint. Para obter informações adicionais sobre a ferramenta, confira [migrar para o SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)
