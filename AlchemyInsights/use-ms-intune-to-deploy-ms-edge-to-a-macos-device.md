# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Usar o Microsoft Intune para implantar o Microsoft Edge em um dispositivo macOS

Para implantar, configurar, monitorar e proteger um aplicativo em um dispositivo macOS, você deve adicionar o aplicativo ao Microsoft Intune. O aplicativo deve ser de um tipo compatível com o Intune, e as opções de gerenciamento disponíveis no Intune dependerão do tipo de aplicativo. O uso do Intune também significa que você não precisará usar a ferramenta de disposição do aplicativo macOS.

O Intune oferece suporte ao Microsoft Edge. Para saber mais, confira [Adicionar o Microsoft Edge aos dispositivos MacOS usando o Microsoft Intune](https://go.microsoft.com/fwlink/?linkid=2134949).
