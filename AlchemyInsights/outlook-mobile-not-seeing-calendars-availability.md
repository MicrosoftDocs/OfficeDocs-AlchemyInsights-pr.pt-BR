# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>Não estou vendo os calendários esperados ao escolher minha disponibilidade

Se você não estiver vendo os calendários esperados, acesse calendário, toque no botão menu no canto superior esquerdo da tela e verifique se os calendários que você espera ver estão selecionados. Somente os calendários habilitados neste menu serão exibidos quando escolher horas para disponibilidade de envio.
