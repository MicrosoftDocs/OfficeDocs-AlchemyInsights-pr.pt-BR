# <a name="increase-microsoft-365-security"></a>Aumentar a segurança do Microsoft 365

Revise a [configuração de segurança recomendada](https://docs.microsoft.com/microsoft-365/security/office-365-security/tenant-wide-setup-for-increased-security?view=o365-worldwide) para seu locatário do Microsoft 365.

- Saiba mais sobre a [Pontuação segura da Microsoft](https://docs.microsoft.com/microsoft-365/security/mtp/microsoft-secure-score?view=o365-worldwide) e [Execute a ação](https://docs.microsoft.com/microsoft-365/security/mtp/microsoft-secure-score?view=o365-worldwide#take-action-to-improve-your-score) para **melhorar sua pontuação**.

- Saiba mais sobre como [responder a incidentes de segurança](https://docs.microsoft.com/microsoft-365/security/office-365-security/office365-security-incident-response-overview?view=o365-worldwide), incluindo **contas comprometidas**.

Para obter mais informações, consulte [introdução ao Microsoft 365 Security](https://docs.microsoft.com/microsoft-365/security/office-365-security/security-roadmap?view=o365-worldwide).
