# <a name="access-denied-banned-sender"></a>Acesso negado, proibidos remetente

 **Erro**: erro de SMTP do servidor remoto para RCPT TO command, host: *xxxx* -com.mail.protection.outlook.com ( *xxx.xxx* ) motivo: 550 5.7.511 acesso negado, proibido remetente [ *xx.xxx.x.xxx* ]. 
  
Se você encontrar esse erro ao enviar email para um destinatário do Office 365, remova a conta afetada na lista de remetentes bloqueados na [https://sender.office.com](https://sender.office.com).
