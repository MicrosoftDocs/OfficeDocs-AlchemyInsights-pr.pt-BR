# <a name="dashboard-not-loading"></a>O Painel não está carregando.

**Sintoma:**

O painel pode estar preso no carregamento ou pode mostrar uma mensagem de erro genérica: "Algo deu errado. Verifique detalhes técnicos para obter mais detalhes."

**Guia de Solução de problemas:**

Confira [Painel UCI não está carregando depois da versão 2003.5](https://support.microsoft.com/help/4558635/uci-dashboard-not-loading-after-the-2003-5-release).
