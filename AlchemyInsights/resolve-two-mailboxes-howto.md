# <a name="a-user-has-two-mailboxes"></a>Um usuário tem duas caixas de correio

Ambientes híbridos que usam o Azure Active Directory Connect (AAD Connect) ou dirSync podem fazer com que um usuário tenha duas caixas de correio: uma na organização local e outra na nuvem.
  
Para resolver esse problema, confira [apagar permanentemente as informações anteriores da caixa de correio](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
