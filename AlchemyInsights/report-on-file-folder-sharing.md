# <a name="report-on-file-and-folder-sharing-in-sharepoint-sites"></a>Relatório sobre compartilhamento de arquivos e pastas em sites do SharePoint

Você pode criar um arquivo CSV detalhando cada arquivo exclusivo, usuário, permissão e link em um determinado site do SharePoint ou biblioteca do OneDrive. Isso pode ajudá-lo a entender como o compartilhamento está sendo usado e se algum arquivo ou pasta está sendo compartilhado com os convidados.

Para obter mais informações, [relate o compartilhamento de arquivos e pastas em um site do SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> No momento, este recurso está implantado para os locatários de primeiro lançamento e será distribuído gradualmente para todos os locatários de produção.
