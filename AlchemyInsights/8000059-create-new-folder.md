# <a name="create-a-new-folder"></a>Criar uma nova pasta

1. Em **pastas** no painel de navegação, selecione a **nova pasta**. 
    
2. Digite um nome para a pasta e pressione Enter.
    
Para saber mais, consulte [criar uma pasta no Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
