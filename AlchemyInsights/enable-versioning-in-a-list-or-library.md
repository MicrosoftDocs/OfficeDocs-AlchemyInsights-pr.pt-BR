# <a name="enable-versioning-for-a-list-or-library"></a>Habilitar o controle de versão para uma lista ou biblioteca

1. Vá para a lista ou biblioteca.
    
2. Clique em **configurações**e, em seguida, clique em **configurações de biblioteca** ou configurações de **lista**.
    
3. Clique em **configurações de controle de versão**.
    
4. Selecione as configurações desejadas e clique em **OK**.
