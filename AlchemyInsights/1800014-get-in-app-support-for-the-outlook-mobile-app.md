# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Obtenha suporte no aplicativo para o aplicativo móvel do Outlook

1. No Outlook app móvel, abra o **Menu**![botão Menu o](media/265b9089-9630-42dd-a244-d9a412d8fe47.png).
    
2. Selecione o ponto de interrogação![botão Ajuda a](media/3b8cbf5a-6ced-4d79-b53c-fa82045c3e25.png) para abrir o **Ajuda &amp; comentários** tela. 
    
3. **Contate o suporte** de toque e explicam o seu problema. 
    
4. Toque em **Enviar** e você receberá uma notificação-app quando um agente de suporte de contatos. 
    
Para obter instruções detalhadas, consulte [obter ajuda no aplicativo do Outlook para iOS e Android](https://support.office.com/article/https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
