# <a name="troubleshoot-issues-with-microsoft-intune"></a>Solucionar problemas com o Microsoft Intune

Analise os recursos listados abaixo para resolver o problema agora.
  
1. Use [Intune Portal de solução de problemas](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) para diagnosticar e resolver alguns falhas comuns. Leia [Este](https://docs.microsoft.com/intune/help-desk-operators) documento para obter mais detalhes.  
2. Revise [Intune TechNet ](https://social.technet.microsoft.com/forums/home?forum=microsoftintuneprod)para obter respostas e soluções para problemas comuns.  
3. Confira a [Integridade do serviço](https://portal.office.com/AdminPortal/Home#/servicehealth) para ver o status do serviço.   
4. Revise [Intune documentação](https://docs.microsoft.com/intune/) para o planejamento, instruções, artigos e mais de solução de problemas.
