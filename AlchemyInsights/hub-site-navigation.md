# <a name="sharepoint-hub-site-navigation"></a>Navegação de site do Hub do SharePoint

Na parte superior de um site do Hub e de todos os sites associados, acima da navegação do site, é a barra de navegação do site do Hub. Se você for o proprietário do site do Hub, poderá personalizar a [barra de navegação](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav). 

**Observação:** É recomendável selecionar um site de comunicação ou um site de equipe usando o modelo moderno como seu site de Hub. Se você usar um site de equipe clássico, a navegação de Hub aparecerá somente nas páginas modernas, e as configurações do site do hub só aparecerão na página moderna.
