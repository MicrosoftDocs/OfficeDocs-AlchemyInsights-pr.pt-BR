# <a name="project-for-the-web-common-issues-and-resolutions"></a>Problemas e resoluções comuns do Project para a Web

Veja a seguir alguns tópicos solicitados frequentemente no Project para a Web:

- Saiba mais sobre o [Project para a Web](https://support.microsoft.com/pt-BR/office/what-is-project-for-the-web-c19b2421-3c9d-4037-97c6-f66b6e1d2eb5), disponível em todas as [três assinaturas do Project](https://products.office.com/project/compare-microsoft-project-management-software).
- [Ative ou desative o Project para a Web ou o Roteiro para os usuários da sua organização](https://docs.microsoft.com/project-for-the-web/turn-project-for-the-web-off). Os administradores podem controlar o acesso ao Project para a Web por meio da página de **Configurações do Project** no Centro de Administração da Microsoft.
- Saiba mais sobre [o que você pode fazer](https://support.office.com/article/what-can-you-do-with-project-for-the-web-b30f5442-be5f-43d2-9072-c95bff778ea1) com o Project para a Web.
- Ajude os usuários a criar seus projetos com o [treinamento do Project para a Web](https://support.office.com/article/get-started-with-project-for-the-web-50bf3e29-0f0d-4b7a-9d2c-7c78389b67ad).
- Familiarize-se com os [limites e restrições](https://docs.microsoft.com/project-for-the-web/project-for-the-web-limits-and-boundaries) do Project para a Web.

Para obter mais informações sobre o Project para a Web, confira a [Ajuda para administradores do Project para a Web](https://docs.microsoft.com/project-for-the-web/projectforweb-admin-home).
