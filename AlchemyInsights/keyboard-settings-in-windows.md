# <a name="keyboard-settings-in-windows-10"></a>Configurações de teclado no Windows 10

**Configurações de tipo**

Para alterar as configurações do Windows relacionadas à verificação ortografia, correção automática, sugestões e outros aspectos do hardware e do teclado por toque, acesse Configurações > **Dispositivos > Digitação**. 

**Configurações de idioma e teclado**

Para alterar as configurações relacionadas a qual teclado ou método de entrada usar para um idioma do Windows, vá para Configurações > Hora **& Idioma > Idioma**. Selecione idiomas na lista **Idiomas preferenciais** e clique em **Opções** para adicionar/remover teclados ou métodos de entrada (IMEs).
