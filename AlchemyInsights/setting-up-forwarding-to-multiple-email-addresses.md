# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Configurando o encaminhamento para vários endereços de email

Se você precisar configurar o encaminhamento de email para um grupo de usuários, crie uma lista de distribuição, adicione vários usuários a ela e configure o encaminhamento para essa lista. Primeiro, no centro de administração, **navegue para grupos de** > [Groups](https://portal.office.com/adminportal/home#/groups)grupos.
  
1. Criar um grupo.

2. Adicionar vários usuários a ele.

3. Configurar o encaminhamento de caixa de correio para esse grupo.
