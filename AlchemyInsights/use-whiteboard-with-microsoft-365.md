# <a name="use-whiteboard-with-microsoft-365"></a>Uso do Whiteboard com o Microsoft 365

O Microsoft Whiteboard é uma tela digital e de forma livre em que pessoas, conteúdo e ideias se unem. 

- [Habilitar Whiteboard](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_07) como um administrador do Microsoft 365. 

- Usar o [Whiteboard com o Microsoft Teams](https://support.microsoft.com/pt-BR/office/use-whiteboard-in-microsoft-teams-7a6e7218-e9dc-4ccc-89aa-b1a0bb9c31ee). 

- [Colaborar](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_27) com outras pessoas no quadro de comunicações. 

Para saber mais, confira [Ajuda do Microsoft Whiteboard](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920).
