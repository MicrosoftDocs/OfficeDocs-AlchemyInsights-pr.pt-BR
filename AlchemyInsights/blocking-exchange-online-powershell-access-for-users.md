# <a name="blocking-exchange-online-powershell-access-for-users"></a>Bloqueando o acesso do PowerShell do Exchange Online para usuários
Como administrador, você tem a capacidade de bloquear o acesso ao PowerShell do Exchange Online para seus usuários. Para obter mais informações, consulte estes tópicos:

- [Habilitar ou desabilitar o acesso ao PowerShell do Exchange Online](https://docs.microsoft.com/powershell/exchange/exchange-online/disable-access-to-exchange-online-powershell)

- [Regras de acesso para cliente no Exchange Online](https://technet.microsoft.com/library/mt842508.aspx) (procure o protocolo **RemotePowerShell** ) 

**Observação**: como descrito no tópico, tenha cuidado para não bloquear o acesso ao PowerShell para administradores do Exchange Online.
