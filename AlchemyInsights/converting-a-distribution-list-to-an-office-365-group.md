# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Converter um grupo de distribuição em um grupo do Microsoft 365

Você pode converter grupos de distribuição em Grupos do Microsoft 365 se eles atenderem a determinados critérios. Para converter um grupo ou solucionar problemas de conversão, consulte [Upgrade distribution lists to Microsoft 365 Groups in Outlook](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).

Também temos um script que você pode executar para ajudá-lo a validar a qualificação do Grupo de Distribuição para atualização para [o Grupo do Microsoft 365.](https://aka.ms/DLToM365Group)
