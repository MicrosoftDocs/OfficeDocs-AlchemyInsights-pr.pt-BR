# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Solucionando problemas de configuração do Outlook Mobile para uma caixa de correio compartilhada

Não é possível acessar a caixa de correio compartilhada do aplicativo Outlook que você instalou no dispositivo móvel. Quer esse recurso? Informe a equipe de engenharia do Outlook. [Vote aqui](https://go.microsoft.com/fwlink/?linked=862116)!
  
Você só pode usar uma caixa de correio compartilhada em um dispositivo móvel abrindo seu navegador, entrando e acessando o Outlook na Web. Para obter instruções, consulte [Adicionar uma caixa de correio compartilhada no Outlook na Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Em Outlook na Web, você poderá acessar a caixa de correio compartilhada.
  
Saiba mais: [obter ajuda no aplicativo para o Outlook para IOS e Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
