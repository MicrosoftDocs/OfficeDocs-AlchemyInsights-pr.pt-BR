# <a name="public-folder-access-from-mobile-devices"></a>Acesso a pastas públicas de dispositivos móveis

As pastas públicas não estão acessíveis a partir de dispositivos móveis.
  
Se você quiser ter funcionalidade semelhante a pastas públicas e acessá-la de dispositivos móveis, consulte [saiba mais sobre os grupos do Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para uma melhor alternativa.
