# <a name="public-folder-access-from-mobile-devices"></a>Acesso a pastas públicas de dispositivos móveis

O acesso a pastas públicas não funciona usando dispositivos móveis, use o OWA (Outlook na Web) ou os clientes do Outlook para Windows na área de trabalho ou Outlook para Mac para acessar pastas públicas.

Se você quiser ter funcionalidade semelhante a pastas públicas e ter conteúdo acessível em dispositivos móveis, consulte [saiba mais sobre os grupos do Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para uma melhor alternativa.
