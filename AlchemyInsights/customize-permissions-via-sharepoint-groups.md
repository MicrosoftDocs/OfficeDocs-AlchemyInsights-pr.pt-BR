# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Como personalizar as permissões de site do SharePoint por meio de grupos do SharePoint 

Um grupo do SharePoint é um conjunto de usuários que têm o mesmo conjunto de permissões para sites e conteúdo. Em vez de atribuir permissões uma pessoa por vez, você pode usar grupos para atribuir convenientemente o mesmo nível de permissão a muitas pessoas de uma só vez.

[Personalizar as permissões de site do SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
