# <a name="creating-public-folders"></a>Criar pastas públicas

Para que os usuários possam usar pastas públicas no Office 365, será necessário criar pelo menos uma caixa de correio de pasta pública. Para criar:
  
1. No portal do O365, usando o menu de navegação à esquerda, navegue até **central** \> de administração do **Exchange**.

2. Navegue até **pastas** \> públicas **caixas de correio de pasta pública**e clique em **Adicionar**.

3. Na página **Caixa de Correio de Pasta Pública**, insira um nome para a caixa de correio de pasta pública.

4. Clique em **Salvar**.

Após a criação de uma caixa de correio, você pode começar a criar pastas individuais ou instruir os usuários a [começar a criar e usar pastas públicas no Outlook](https://support.office.com/article/Create-and-share-a-public-folder-in-Outlook-a2835011-d524-4a5c-a207-05c159bb2a97).
