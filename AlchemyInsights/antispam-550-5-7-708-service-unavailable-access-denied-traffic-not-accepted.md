# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 acesso negado, tráfego não aceito deste IP

Esse erro ocorre ao enviar email de endereços IP conhecidos e de baixa reputação que são normalmente usados por novos clientes.

Se você precisar enviar emails desses endereços IP antes de adquirir licenças, entre em contato com o suporte para solicitar uma exceção até que você possa comprar licenças.
