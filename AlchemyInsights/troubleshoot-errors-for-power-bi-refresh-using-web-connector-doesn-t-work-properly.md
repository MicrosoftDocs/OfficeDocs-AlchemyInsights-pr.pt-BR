# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Atualização usando o Web Connector não funciona corretamente

Se você tiver um script do Web Connector que está usando a função [Web. Page](https://msdn.microsoft.com/library/mt260924.aspx) e tiver atualizado seu conjunto de trabalho ou relatório após 18 de novembro de 2016, será necessário usar um gateway para que a atualização funcione corretamente.

Para obter mais informações: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
