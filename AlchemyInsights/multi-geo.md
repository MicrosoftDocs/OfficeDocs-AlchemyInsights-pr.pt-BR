# <a name="microsoft-365-multi-geo"></a>Microsoft 365 multigeográfico

Com o Microsoft 365 multigeográfico, sua organização pode expandir sua presença da Microsoft 365 para várias regiões geográficas e/ou países dentro de seu locatário existente. Confira os links abaixo para obter mais informações.

- [Microsoft 365 multigeográfico](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Recursos multigeográficos no OneDrive e SharePoint Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-onedrive-and-sharepoint-online-in-office-365)
- [Recursos multigeográficos no Exchange Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-exchange-online)
- [Plano para o Microsoft 365 multigeo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Configurar várias GEOS](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
