# <a name="create-a-shared-mailbox"></a>Criar uma caixa de correio compartilhada

Uma caixa de correio compartilhada permite que mais de uma pessoa enviem e recebam email para um alias, como sales@contoso.com, sem atribuir outra licença. Para criar uma caixa de correio compartilhada, vá para o [Guia de Email compartilhadas](https://portal.office.com/adminportal/home).
  
Consulte o artigo [criar uma caixa de correio compartilhada](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) para dicas e mais informações sobre como usar uma caixa de correio compartilhada.
