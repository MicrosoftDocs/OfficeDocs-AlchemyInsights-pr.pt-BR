# <a name="create-a-shared-mailbox"></a>Criar uma caixa de correio compartilhada

Uma caixa de correio compartilhada permite que mais de uma pessoa envie e receba emails para um alias, como o sales@contoso.com, sem atribuir outra licença. 

Confira o artigo [criar uma caixa de correio compartilhada](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) para obter dicas e mais informações sobre como usar uma caixa de correio compartilhada.
