As etapas para adicionar usuários externos/convidados podem ser diferentes dependendo do que você precisa para acessar o. Escolha uma das seguintes opções:
  
- [Visão geral do compartilhamento externo do SharePoint Online](https://docs.microsoft.com/sharepoint/external-sharing-overview)

- [Acesso de convidados nos grupos do Microsoft 365](https://support.office.com/article/guest-access-in-office-365-groups-bfc7a840-868f-4fd6-a390-f347bf51aff6)

- [Acesso para convidado no Microsoft Teams](https://docs.microsoft.com/microsoftteams/guest-access-checklist)
