# <a name="getting-an-exchange-hybrid-key"></a>Obtendo uma chave de híbrida do Exchange

Para conectar sua organização local para o Exchange Online no Office 365, você precisará configurar pelo menos um servidor do Exchange em sua organização local. Esse servidor é usado para criar uma implantação híbrida que se conecta a sua organização do Exchange no local para o Exchange Online. Para obter uma chave híbrida para esse servidor, siga as etapas no [Assistente de distribuição de chaves de produto do Exchange híbrida](http://aka.ms/hybridkey).
