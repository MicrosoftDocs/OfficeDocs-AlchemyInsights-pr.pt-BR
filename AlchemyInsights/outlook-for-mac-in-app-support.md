# <a name="in-app-support-in-outlook-for-mac"></a>Suporte no aplicativo no Outlook para Mac

Para receber suporte no-app para problemas do Outlook para Mac, abra o Outlook **e clique em** \> **contatar o suporte**. Digite sua pergunta e um membro da equipe de suporte do Outlook para Mac entrará em contato com você. 

Saiba mais: [contatar o suporte no Outlook para Mac](https://support.office.com//article/d0410177-8e65-4487-93f7-206a3a3d71a8)
