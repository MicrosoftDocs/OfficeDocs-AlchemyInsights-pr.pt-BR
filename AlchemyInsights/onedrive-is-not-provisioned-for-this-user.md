# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive não está provisionado para este usuário

Para contas recém-criado, os usuários precisam entrar no portal do Office 365 com sua própria conta e clique em OneDrive para iniciar manualmente a provisão do site OneDrive.
  
No entanto, se o usuário clicou já OneDrive e seu site pessoal não está provisionado dentro de 24 horas, efetue um tíquete de suporte para obter assistência.
