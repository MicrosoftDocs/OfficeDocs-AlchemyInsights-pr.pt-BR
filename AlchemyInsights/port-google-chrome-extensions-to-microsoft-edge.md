# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Porta Google Chrome Extensions para o Microsoft Edge (Chromium)

É fácil [portar as extensões Google Chrome para o Microsoft Edge (Chromium)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). Na maioria dos casos, apenas as alterações mínimas são necessárias para executar essas extensões no Microsoft Edge.

As APIs de extensão e as chaves de manifesto compatíveis com o Google Chrome são compatíveis com código com o Microsoft Edge. No entanto, o Microsoft Edge não é compatível com as APIs de extensão Chrome. gcm, Chrome. Identity. getaccounts, Chrome. Identity. getAuthToken e Chrome. instanceID.
