# <a name="skype-icon-missing"></a>Ícone do Skype ausente

Removemos temporariamente o ícone do Skype do Outlook.com como parte de uma atualização de design e atualização de recursos Estamos entusiasmados a ser distribuído. Agradecemos sua paciência enquanto preparamos uma experiência nova e aprimorada para você.
