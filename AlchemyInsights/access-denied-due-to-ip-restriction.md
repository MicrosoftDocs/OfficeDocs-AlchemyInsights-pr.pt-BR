# <a name="access-denied-due-to-ip-restriction"></a>Acesso negado devido à restrição de IP

Como um administrador de ti, você pode controlar o acesso aos recursos do SharePoint e do OneDrive no Microsoft 365 com base em locais de rede definidos nos quais confia. Isso também é conhecido como política baseada em local. Para obter mais informações, consulte [controlar o acesso com base no local de rede](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location).
