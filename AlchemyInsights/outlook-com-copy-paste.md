# <a name="copy-and-paste-in-outlookcom"></a>Copiar e colar no Outlook.com

Não há botões copiar ou colar no Outlook.com, mas você pode usar atalhos de teclado:

**Em um computador:** CTRL + C para copiar, CTRL + X para recortar e CTRL + V para colar.

**Em um Mac:** CMD + C para copiar, CMD + X para recortar e CMD + V para colar.
