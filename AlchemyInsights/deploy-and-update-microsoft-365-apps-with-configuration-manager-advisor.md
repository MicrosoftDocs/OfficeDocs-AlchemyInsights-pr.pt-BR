# <a name="deploy-and-update-microsoft-365-apps-with-configuration-manager-advisor"></a>Implantar e atualizar aplicativos do Microsoft 365 com o consultor do Configuration Manager

Para organizações que usam o Configuration Manager, você pode usar o consultor Implantar e atualizar aplicativos do [Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2146549) com o Configuration Manager para gerar um script que configurará automaticamente sua implantação do Microsoft 365 Apps usando as práticas recomendadas pelos engenheiros do FastTrack.

Use este guia para criar seus grupos de implantação, personalizar seus aplicativos e recursos do Office, configurar instalações dinâmicas ou enxutas e, em seguida, executar o script para criar aplicativos, regras de implantação automática e coleções de dispositivos que você precisa para direcionar sua implantação.
