# <a name="sharepoint-libraries-mapped-to-network-drives-become-disconnected"></a>Bibliotecas do SharePoint mapeadas para unidades de rede são desconectadas

O mapeamento de uma biblioteca de documentos do SharePoint para uma unidade de rede é temporário. Ocasionalmente, você deve abrir o site do SharePoint no Internet Explorer e selecionar **permanecer conectado** para impedir que a sessão expire. 
  
[Obter ajuda para configurar e solucionar problemas de unidades de rede mapeadas](https://support.office.com/article/ef399c67-4578-4c3a-adbe-0b489084eabe.aspx).
  
Para trabalhar com uma biblioteca no explorador de arquivos, [sincronize os arquivos do SharePoint com o novo cliente de sincronização do onedrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) , que inclui [arquivos sob demanda](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Acessar todos os seus arquivos no OneDrive sem usar espaço de armazenamento local.
