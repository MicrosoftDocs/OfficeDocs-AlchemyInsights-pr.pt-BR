# <a name="site-is-not-available"></a>Site não está disponível

Se o seu site do SharePoint ou OneDrive não está disponível para usuários múltiplos, pode haver um problema temporário do serviço. Verifique se o[Painel de integridade do serviço](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver se o seu locatário é afetado.
