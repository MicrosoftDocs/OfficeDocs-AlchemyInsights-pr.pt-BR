# <a name="export-user-account-information"></a>Exportar informações de conta de usuário

Para exportar informações de conta de usuário no Centro de administração do Microsoft 365, incluindo informações sobre licenças, política de senha, contas bloqueadas, status de sincronização do AAD e muito mais, acesse a página Usuários ativos e selecione  >  [](https://go.microsoft.com/fwlink/p/?linkid=834822) **Exportar Usuários**.
