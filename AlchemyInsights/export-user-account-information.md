# <a name="export-user-account-information"></a>Exportar informações de conta de usuário

Para exportar as informações da conta de usuário no centro de administração do Microsoft 365, incluindo informações sobre licenças, política de senha, contas bloqueadas, status de sincronização do AAD e mais, acesse a página usuários[ativos](https://go.microsoft.com/fwlink/p/?linkid=834822) do **usuário** > e selecione **exportar Usuários**.
