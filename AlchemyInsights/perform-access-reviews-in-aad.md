# <a name="perform-access-reviews-in-azure-active-directory"></a>Executar revisões de acesso no Azure Active Directory

Os revisores designados podem realizar revisões de acesso do Azure Active Directory de associações de grupo, acesso de aplicativo e atribuições de função. Essas revisões ajudam a reduzir os riscos de segurança que sua organização pode enfrentar quando os usuários têm mais acesso do que precisam. Você pode executar revisões de acesso usando [meus aplicativos](https://go.microsoft.com/fwlink/?linkid=2134605) ou [meu acesso](https://go.microsoft.com/fwlink/?linkid=2134505).
