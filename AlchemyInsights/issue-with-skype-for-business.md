# <a name="issue-with-skype-for-business"></a>Problema com Skype para os negócios?

Podemos foi detectado que os seguintes domínios de tem problemas que podem impedir que Skype para negócios funcionando adequadamente:
  
{HealthCheckString}
  
Os registros SRV, CNAME ou NS das configurações de domínio (DNS) não estão configurados corretamente para Skype para negócios. Você pode consultar a [registros de DNS atualizando no Office 365](https://support.office.com/article/https://support.office.com/article/Create-DNS-records-for-Office-365-when-you-manage-your-DNS-records-B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23.aspx) para obter instruções detalhadas. 
  
Para corrigi-lo, vá para a página de [domínios](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/Domains) no Centro de administração do Office 365 e vamos ajudá-lo com ele. 
  
Se você estiver tendo problemas para se comunicar com usuários que são externos à sua organização, você pode consultar a [Configurar Skype para comunicações externas Business Online](https://support.office.com/article/https://support.microsoft.com/help/10041/set-up-skype-for-business-online-external-communications.aspx) para obter instruções detalhadas. 
  
[Verifique o FAQ domínios](https://support.office.com/article/https://support.office.com/article/7b7b075d-79f9-4e37-8a9e-fb60c1d95166.aspx) outras soluções comuns.
