# <a name="yammer---possible-service-issue"></a>Yammer-possível problema de serviço

Pode haver um problema temporário no qual os usuários podem enfrentar erros ao acessar o Yammer. Verifique o [painel de integridade do serviço](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver se isso afeta sua rede.
