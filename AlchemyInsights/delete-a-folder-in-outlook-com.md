# <a name="delete-a-folder"></a>Excluir uma pasta

1. No painel esquerdo, clique com o botão direito do mouse na pasta ou subpasta que você deseja excluir e selecione **excluir**. 
    
2. Selecione **OK** para confirmar. 
    
Para saber mais, confira [excluir uma pasta no Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
