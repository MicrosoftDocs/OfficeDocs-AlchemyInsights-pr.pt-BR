# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Usar SaRA para diagnosticar e resolver problemas de autenticação do Outlook

**Observação**: Verifique se [os padrões de segurança](https://aka.ms/securitydefaults) estão habilitados para sua organização. Se o seu locatário tiver sido criado após outubro de 21 de 2019 e o Outlook estiver solicitando uma senha repetidamente, você poderá ter **os padrões de segurança** habilitados no locatário.

É altamente recomendável que você use o Outlook para resolver problemas de diagnóstico de [minha senha](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) na máquina afetada, em que o Outlook solicita uma senha continuamente. Esse diagnóstico de [SaRA](https://diagnostics.office.com/#/) faz verificações automáticas e retorna possíveis soluções para que você use o para resolver problemas detectados.
