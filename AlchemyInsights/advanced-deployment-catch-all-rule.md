# <a name="deploy-office-apps"></a>Implantar aplicativos do Office

Para implantar aplicativos do Office para vários usuários finais por meio da ferramenta de implantação do Office, System Center Configuration Manager, como parte de uma imagem de disco do sistema operacional ou em um cenário de computador compartilhado usando o Windows Server com serviços de área de trabalho remota (RDS) ou terminal Serviços, considere o uso do guia passo a passo no **supervisor de implantação do Office 365 ProPlus**.
  
Para começar, verifique se você está conectado como administrador de locatários ou administrador global e clique aqui: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
