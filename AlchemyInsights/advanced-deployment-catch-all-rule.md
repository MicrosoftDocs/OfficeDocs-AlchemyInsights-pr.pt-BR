# <a name="deploy-office-apps"></a>Implantar aplicativos do Office

Para implantar aplicativos do Office para vários usuários finais por meio da ferramenta de implantação do Office, o Gerenciador de configuração do Microsoft EndPoint, como parte de uma imagem de disco do sistema operacional, ou em um cenário de computador compartilhado usando o Windows Server com serviços de área de trabalho remota (RDS) ou serviços de terminal, considere usar o guia passo a passo no **Microsoft 365 Apps for Enterprise Deployment Advisor**.
  
Para começar, verifique se você está conectado como administrador de locatários ou administrador global e clique aqui: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
