# <a name="deploy-office-apps"></a>Implantar o Office apps

Para implantar aplicativos do Office para vários usuários finais por meio da ferramenta de implantação do Office, System Center Configuration Manager, como parte de uma imagem de disco do sistema operacional ou em um cenário de computador compartilhado usando o Windows Server com serviços de área de trabalho remota (RDS) ou de Terminal Serviços, considere usar o guia passo a passo no **Office 365 ProPlus Supervisor de implantação**.
  
Para começar, certifique-se de que você está conectado como um administrador proprietário ou administrador global e, em seguida, clique aqui: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
