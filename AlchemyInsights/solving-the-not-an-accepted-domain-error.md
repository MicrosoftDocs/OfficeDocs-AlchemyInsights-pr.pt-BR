# <a name="got-a-parameter-cannot-be-found-error"></a>Recebeu um erro "não foi possível encontrar o parâmetro"?

Se você receber a mensagem de erro **"não é possível localizar um parâmetro que corresponda ao nome de parâmetro" EmailAddresses "**, isso significa que está demorando um pouco mais para o Microsoft 365 para concluir a configuração do seu locatário ou do seu domínio personalizado, se você tiver adicionado um.
  
O processo de configuração pode levar até quatro horas para ser concluído. Aguarde um pouco até ele terminar e tente novamente.
