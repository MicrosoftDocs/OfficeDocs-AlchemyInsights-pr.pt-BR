# <a name="account-is-restricted-for-sending-too-much-spam"></a>A conta é restrita por enviar muito spam

**Observação** Acesse o Centro de segurança do Microsoft 365 em [https://security.microsoft.com](https://security.microsoft.com).

Como administrador, se um dos seus usuários estiver restrito a enviar emails, você poderá [desbloquear a conta](https://security.microsoft.com/?hash=/restrictedusers) **sem abrir um caso de suporte**. O usuário poderá enviar emails dentro de uma hora após remover a restrição.

O serviço impede que um usuário envie emails após detectar evidências de uma conta comprometida e/ou spam de saída. Como precaução, siga as etapas para [Responder a uma conta de email comprometida](https://docs.microsoft.com/microsoft-365/security/office-365-security/responding-to-a-compromised-email-account) para o usuário.
