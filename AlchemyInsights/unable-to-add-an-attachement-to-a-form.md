# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Não é possível adicionar um anexo a um formulário de face externa

Se você estiver tentando adicionar um anexo a uma pesquisa externa do Microsoft Forms, esse recurso ainda não estará disponível. Para acompanhar o status dessa solicitação de recurso, [acesse esta entrada UserVoice](https://go.microsoft.com/fwlink/?linkid=2133069)e adicione seu voto.
