# <a name="what-are-groups"></a>O que são Grupos?

Os Grupos do Microsoft 365 permitem que você escolha um conjunto de pessoas com as quais deseja colaborar e facilmente configurar uma coleção de recursos para essas pessoas compartilharem. Você não precisa se preocupar em atribuir permissões manualmente a todos esses recursos, pois a adição de membros ao grupo fornece automaticamente as permissões necessárias às ferramentas que seu grupo fornece. Além disso, os grupos são a experiência nova e aprimorada para o que usamos para usar listas de distribuição ou caixas de correio compartilhadas para fazer.  Confira [Saiba mais sobre os Grupos do Microsoft 365.](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)
