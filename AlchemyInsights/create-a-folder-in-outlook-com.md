# <a name="create-a-folder"></a>Criar uma pasta

1. No painel esquerdo, selecione **criar nova pasta** ao lado de **pastas**. 
    
2. Na caixa nova pasta, digite um nome para a pasta e pressione Enter.
    
Para saber mais, confira [criar uma pasta no Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
