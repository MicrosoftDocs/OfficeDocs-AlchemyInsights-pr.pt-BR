# <a name="managing-microsoft-stream-licenses"></a>Gerenciando licenças do Microsoft Stream

**Importante:** Você precisa estar na função de administrador de locatários para executar operações descritas neste tópico.

Você precisa ser um administrador global ou um administrador de gerenciamento de usuários. Para obter mais informações, consulte [sobre as funções de administrador do Office 365](https://support.office.com/article/About-Office-365-admin-roles-da585eea-f576-4f55-a1e0-87090b6aaa9d).

Você pode atribuir [licenças a contas de usuário com o Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
