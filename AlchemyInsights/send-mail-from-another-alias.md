# <a name="send-email-from-an-alias-or-secondary-address"></a>Enviar email a partir de um alias ou o endereço secundário

O Office 365 atualmente não oferece suporte para enviar e-mails de um alias ou o endereço de email secundário. Alias de email podem ser usados para receber emails apenas. Se você deseja enviar email a partir de outro endereço de email, você precisará criar uma caixa de correio compartilhada. Caixa de correio compartilhada fornecerá a você outro endereço de email, sem exigir a outra licença. 
  
- [Clique aqui para criar uma caixa de correio compartilhada](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)
    
Você também pode ler o artigo [criar uma caixa de correio compartilhada](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9) que dirá tudo o que você precisa saber sobre caixas de correio compartilhadas, incluindo como usá-los.
