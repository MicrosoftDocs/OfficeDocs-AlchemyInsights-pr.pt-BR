# <a name="duplicate-object-synchronization-errors"></a>Erros de sincronização do objeto duplicado

Se você receber um dos seguintes erros durante uma sincronização:
  
- Já existe um objeto sincronizado com o mesmo endereço de proxy em seu diretório do Microsoft Online Services.
    
- Uma mensagem semelhante para UserPrincipalName.
    
Use a integridade de AADConnect novo diagnóstico para resolver erros do objeto duplicado. Para obter mais informações, consulte [diagnosticar e corrigir erros de sincronização de atributo duplicada](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
