# <a name="how-to-move-a-onenote-notebook"></a>Como mover um bloco de anotações do OneNote

* Para mover um bloco de anotações do OneNote para o onedrive, vá para compartilhamento de **arquivos**  >  **Share**  >  **mover blocos de anotações** no OneNote 2016. Consulte [mover um bloco de anotações do OneNote para o onedrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082) para obter mais informações.

* Para obter informações adicionais sobre as práticas recomendadas ao se preparar para mover blocos de anotações, confira [práticas recomendadas de sincronização do OneNote](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices).

* [Mover um bloco de anotações do OneNote para Windows que você compartilhou com outras pessoas](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) oferece mais informações sobre como mover blocos de anotações compartilhados.
