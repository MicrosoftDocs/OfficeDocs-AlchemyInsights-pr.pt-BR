# <a name="prevent-teams-from-starting-automatically"></a>Impedir que o Teams seja iniciado automaticamente

Por padrão, o Microsoft Teams é aberto automaticamente quando você inicia o dispositivo.

Para alterar essa configuração, selecione a imagem de perfil do Microsoft Teams e, em seguida, selecione **configurações**  >   **gerais**. Em  **aplicativo**, desmarque  **aplicativo de início automático**.

Para obter mais informações, consulte  [alterar configurações no Teams](https://support.microsoft.com/office/b506e8f1-1a96-4cf1-8c6b-b6ed4f424bc7).
