# <a name="manage-who-can-create-office-365-groups"></a>Gerenciar quem pode criar Grupos do Office 365

Por padrão, qualquer pessoa em sua organização pode criar grupos de O365.
  
Se desejar alterar isso, você pode criar um grupo de segurança com o PowerShell que limita a criação de grupo a um subconjunto de usuários.
  
Siga as instruções passo a passo aqui: [Gerenciar quem pode criar grupos do Office 365](https://support.office.com/article/4c46c8cb-17d0-44b5-9776-005fced8e618)
