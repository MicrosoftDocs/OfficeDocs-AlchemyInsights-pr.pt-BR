# <a name="manage-who-can-create-microsoft-365-groups"></a>Gerenciar quem pode criar grupos do Microsoft 365

Por padrão, qualquer pessoa em sua organização pode criar grupos do O365.
  
Se quiser alterar isso, você pode criar um grupo de segurança com o PowerShell que limita a criação de grupo a um subconjunto de usuários.
  
Siga as instruções passo a passo aqui: [gerenciar quem pode criar grupos do Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
