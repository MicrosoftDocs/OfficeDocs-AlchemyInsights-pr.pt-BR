# <a name="onedrive-recycle-bin"></a>Lixeira do OneDrive

Quando você exclui um item ou site do OneDrive, ele não é removido imediatamente. Os itens excluídos vão para a lixeira por um período de tempo. Durante esse período, você pode restaurar os itens excluídos para o local original. [Saiba mais](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f?ui=en-US&rs=en-US&ad=US).
