# <a name="domain-status---no-services-selected"></a>Status do Domínio - Nenhum serviço selecionado

Um status de saúde de domínio está disponível depois que você conecta o domínio diretamente Microsoft 365 serviços como Exchange. Para fazer isso, vá para o centro de administração Microsoft 365 e selecione **Registros DNS**  >  **Gerenciar DNS**.

Se você não puder conectar o domínio diretamente aos serviços Microsoft 365, o status de saúde do domínio não estará disponível. Por exemplo, quando você usa o roteamento DNS personalizado para email, serviços de filtragem de spam externo ou está usando o Exchange local e o Exchange Online (também chamado de implantação híbrida).
