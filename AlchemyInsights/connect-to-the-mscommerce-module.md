# <a name="connect-to-the-mscommerce-module"></a>Conectar-se ao módulo MSCommerce

Você deve estar conectado ao módulo MSCommerce antes de poder exibir ou definir a política AllowSelfServicePurchase.  

Para se conectar ao módulo MSCommerce, no prompt do PowerShell (PS C:\), digite o seguinte comando:

    Connect-MSCommerce

Isso abrirá uma caixa de diálogo de entrada. Insira seu nome de usuário e senha para entrar.

**Observação:**&nbsp;&nbsp;a conta usada para entrar deve ser um administrador de empresa ou de cobrança.
