# <a name="create-a-shared-mailbox"></a>Criar uma caixa de correio compartilhada

Uma caixa de correio compartilhada permite que mais de uma pessoa envie e receba emails para um alias, como o **Sales@contoso.com**, sem atribuir outra licença.
  
Consulte [criar uma caixa de correio compartilhada](https://support.office.com/client/871a246d-3acd-4bba-948e-5de8be0544c9) para obter mais informações.
