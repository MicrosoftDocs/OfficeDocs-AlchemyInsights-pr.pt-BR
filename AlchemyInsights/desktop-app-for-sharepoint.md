# <a name="desktop-app-for-sharepoint"></a>Aplicativo de área de trabalho para SharePoint

Sua organização é inserida no programa de visualização privada para o aplicativo de área de trabalho para SharePoint. A Microsoft não suporta atualmente o aplicativo de área de trabalho para o SharePoint (visualização privada). Use seus parceiros de visualização privada da Microsoft para obter assistência com o aplicativo de área de trabalho do SharePoint.
