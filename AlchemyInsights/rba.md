# <a name="about-room-mailboxes"></a>Sobre caixas de correio de sala

Uma caixa de correio de sala é uma caixa de correio de recursos atribuída a um local físico, como uma sala de conferência, auditório ou sala de treinamento. Para obter mais informações sobre caixas de correio de sala, consulte [criar e gerenciar caixas de correio de sala](https://go.microsoft.com/fwlink/p/?linkid=717533).
