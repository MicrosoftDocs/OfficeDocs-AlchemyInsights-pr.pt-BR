# <a name="delays-in-receiving-sharepoint-and-onedrive-alerts"></a>Atrasos no recebimento de alertas do SharePoint e do OneDrive

- Se todos os alertas de vários arquivos ou [](https://portal.office.com/adminportal/home?ref=/servicehealth) bibliotecas se atrasarem, visite o painel de Saúde do Serviço para verificar se há **avisos/incidentes** que possam estar ocorrendo com o SharePoint ou o Exchange.
- Se um alerta individual de um arquivo ou biblioteca específico não **for entregue,** tente excluí-lo e recriá-lo. Consulte [Gerenciar, exibir ou excluir alertas do SharePoint](https://support.microsoft.com/office/99dfb19c-9a90-4a8c-aba1-aa8c8afb0de2) para recriar o alerta.
- Verifique a pasta Lixo Eletrônico ou Spam em seu email.

> [!NOTE]
> - Os alertas não podem ser enviados a um Grupo de Distribuição. Somente os grupos Security e O365 são suportados.
> - Não é possível personalizar modelos de email de alerta. Você deve usar o Microsoft Flow ou o SharePoint Designer Workflow para conseguir isso.
