# <a name="migrating-email-from-imap-email-system-to-office-365"></a>Migrando emails do sistema de email IMAP para o Office 365

Para migrar emails de um sistema de email IMAP, siga este processo: [migração IMAP no centro de administração do Office 365](https://support.office.com/article/4682f2e4-f720-4868-91ab-207f5b0c325d)
  
Muitas organizações grandes precisam planejar com cuidado para gerenciar as expectativas do usuário. O [que você precisa saber sobre a migração de caixas de correio IMAP para o artigo do Office 365](https://docs.microsoft.com/en-us/Exchange/mailbox-migration/migrating-imap-mailboxes/migrating-imap-mailboxes) ajudará você a gerenciar a alteração. 

Verifique também a página [adotar o Office 365 com o FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) para obter ajuda para a migração, se for qualificado.
