# <a name="delete-a-folder-in-outlookcom"></a>Excluir uma pasta no Outlook.com

1. No painel esquerdo, clique com o botão direito do mouse na pasta ou subpasta que você deseja excluir e selecione **Excluir pasta**.
2. Selecione **OK** para confirmar.

Para saber mais, confira [trabalhar com pastas de mensagens no Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
