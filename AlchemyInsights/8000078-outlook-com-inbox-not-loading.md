# <a name="inbox-not-loading-with-firefox"></a>A caixa de entrada não está carregando com o Firefox

Determinados complementos no navegador Firefox podem impedir que sua caixa de entrada do Outlook.com seja carregada.
  
Visite o suporte Firefox para saber como [desabilitar ou remover](https://support.mozilla.org/kb/disable-or-remove-add-ons)Complementos.
