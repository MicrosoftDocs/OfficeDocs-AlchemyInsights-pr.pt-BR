# <a name="add-a-sharepoint-calendar"></a>Adicionar um calendário do SharePoint

Os Calendários do SharePoint são tipos de lista clássicos que às vezes são confundidos com calendários de grupo do Microsoft 365.
 
Para criar um Calendário do SharePoint, siga estas etapas:
 
1.  Selecione o **ícone de engrenagem Configurações** e, em seguida, **selecione Adicionar um aplicativo**.
2.  Na caixa **Encontrar um aplicativo** de pesquisa, digite **Calendário** e pressione Enter.
3.  Selecione o aplicativo intitulado **Calendário** e insira um nome para seu calendário.

Para obter mais informações sobre grupos do Microsoft 365, [consulte Learn about Microsoft 365 Groups](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
