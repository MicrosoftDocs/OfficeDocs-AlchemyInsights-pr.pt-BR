# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>Excluir itens ou esvaziar a Lixeira de um site do SharePoint 

Quando você exclui um item de uma biblioteca de documentos ou lista do SharePoint, ele não é removido imediatamente do SharePoint. Os itens excluídos vão para a Lixeira do site por um período de tempo. Durante esse tempo, você pode restaurar os itens excluídos no local original ou removê-los para liberar espaço de armazenamento.

[Excluir itens ou esvaziar a Lixeira de um site do SharePoint](https://support.office.com/article/2e713599-d13e-40d6-96dc-66f0a366f74e)
