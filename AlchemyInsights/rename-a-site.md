# <a name="rename-a-site"></a>Renomear um site

Para alterar o nome de um site, por exemplo, de https://contoso.sharepoint.com/sites/sample2 para [](https://go.microsoft.com/fwlink/?Linkid=2018691), você precisa criar um novo site e mover o conteúdo do site antigo para o novo. Confira informações sobre como mover itens nas bibliotecas de documentos Mover ou copiar itens de uma biblioteca de documentos do SharePoint.
  
Observação: Se você está tentando alterar o nome de domínio (ou seja, a contoso de parte da URL acima) em uma URL de SharePoint Online, que não é possível. Consulte [alterar seus domínios](https://go.microsoft.com/fwlink/?Linkid=2018696).
