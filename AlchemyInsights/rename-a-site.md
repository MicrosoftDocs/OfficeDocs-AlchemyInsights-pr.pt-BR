# <a name="rename-a-site"></a>Renomear um site

Para alterar o nome de um site, por exemplo, de https://contoso.sharepoint.com/sites/sample1 para https://contoso.sharepoint.com/sites/sample2, você precisa criar um novo site e mova todo o conteúdo do site antigo para o novo. Para informações sobre como mover itens em bibliotecas de documentos, consulte [Mover ou copiar itens em uma biblioteca de documentos do SharePoint](https://go.microsoft.com/fwlink/?Linkid=2018691).
  
Observação: Se você está tentando alterar o nome de domínio (ou seja, a contoso de parte da URL acima) em uma URL de SharePoint Online, que não é possível. Consulte [alterar seus domínios](https://go.microsoft.com/fwlink/?Linkid=2018696).
