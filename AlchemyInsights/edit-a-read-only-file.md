# <a name="edit-a-read-only-file"></a>Editar um arquivo somente leitura

Em bibliotecas de documentos, quando você clica no nome de um documento, o programa do Office pode abri-lo inesperadamente como um arquivo somente leitura. Para abri-lo para edição, clique em **abrir** (os três pontos) e clique em **Editar.**
