# <a name="edit-a-read-only-file"></a>Editar um arquivo somente leitura

Em bibliotecas de documentos, quando você clica em nome de um documento, o programa do Office pode inesperadamente abri-lo como um arquivo somente leitura. Para abri-lo para edição, clique em **Abrir** (nos três pontos) e, em seguida, clique em **Editar.**
