# <a name="changes-to-external-user-access"></a>Alterações para acesso de usuário externo

Iniciando 23 de março de 2018, usuários externos não será capazes de acessar o conteúdo que foi compartilhado com todos, todos os usuários autenticados ou todos os usuários de formulários, mesmo que eles tenham sido convidados ou adicionados aos grupos autorizados. Os usuários externos só poderá acessar o conteúdo que foi compartilhado com o usuário ou grupos aos quais o usuário pertence. Se você deseja manter as configurações existentes, consulte:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
