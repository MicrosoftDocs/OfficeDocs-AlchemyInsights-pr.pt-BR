# <a name="adding-an-overlay-to-a-sharepoint-calendar"></a>Adicionar uma Sobreposição a um calendário do SharePoint

Os Calendários do SharePoint são tipos de lista clássicos que às vezes são confundidos com calendários de grupo do Microsoft 365.
 
Se você usar um calendário do SharePoint em seu site, poderá sobrepor seu próprio calendário do Exchange nele para criar uma exibição de calendário combinada. Como alternativa, você pode sobrepor outro calendário do SharePoint enquanto ambos os calendários do SharePoint estão no mesmo conjunto de sites.
 
Para adicionar uma Sobreposição a um calendário clássico do SharePoint, siga as etapas em Sobrepor um calendário do SharePoint com um calendário do [Exchange ou do SharePoint](https://support.office.com/article/Overlay-a-SharePoint-calendar-with-a-calendar-from-Exchange-or-SharePoint-4CAEBE59-3994-4A94-9322-B31ABB8A5E9A).
