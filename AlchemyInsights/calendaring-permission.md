# <a name="calendar-permissions"></a>Permissões de calendário

Se você estiver tendo problemas para acessar seu calendário ou outros calendários, use o [Assistente de recuperação e suporte da Microsoft para o Office 365](https://diagnostics.office.com/).
  
Para saber mais sobre como compartilhar seu calendário, confira [compartilhar um calendário do Outlook com outras pessoas](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88.aspx).
