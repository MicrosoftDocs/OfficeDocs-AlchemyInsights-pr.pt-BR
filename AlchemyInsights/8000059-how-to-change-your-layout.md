# <a name="how-to-change-your-layout"></a>Como alterar seu layout.

É fácil alterar as configurações no Outlook.com para que se parece e se comporta da maneira que você deseja que ele. Selecione **configurações** na barra de ferramentas para ver suas escolhas.
