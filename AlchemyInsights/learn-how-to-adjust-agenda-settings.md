# <a name="learn-how-to-configure-agenda-settings"></a>Saiba como definir as configurações de agenda

As configurações de agenda estão localizadas nas configurações de **calendário** do Outlook na Web. Para obter mais informações, consulte [configurações de calendário no Outlook na Web](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).

Os administradores podem usar o cmdlet **Set-CalendarNotification** no PowerShell do Exchange Online para definir as configurações de agenda. Para obter mais informações, consulte [Set-CalendarNotification](https://technet.microsoft.com/library/dd351284).
