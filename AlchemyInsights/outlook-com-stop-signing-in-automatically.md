# <a name="how-to-stop-signing-in-automatically"></a>Como parar de entrar automaticamente

Para parar de entrar automaticamente, desmarque a caixa para **me manter conectado** na página onde você digitou sua senha.
  
Para saber mais, confira [como entrar ou sair do Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy).
