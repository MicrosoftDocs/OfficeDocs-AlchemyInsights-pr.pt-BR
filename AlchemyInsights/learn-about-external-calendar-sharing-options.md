# <a name="learn-about-external-calendar-sharing-options"></a>Saiba mais sobre as opções de compartilhamento de calendários externos

Se você precisar agendar reuniões com pessoas de fora da sua organização, o Office 365 permite que você torne calendários disponíveis para "usuários externos", aqueles que precisam ver o horário de disponibilidade, mas não têm contas de usuário para o seu ambiente do Office 365. Para obter mais informações, consulte [compartilhar calendários com usuários externos](https://support.office.com/article/fb00dd4e-2d5f-4e8d-8ff4-94b2cf002bdd.aspx).
