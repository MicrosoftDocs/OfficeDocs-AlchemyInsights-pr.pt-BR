# <a name="learn-about-external-calendar-sharing-options"></a>Saiba mais sobre as opções de compartilhamento de calendários externos

Se você precisar agendar reuniões com pessoas de fora da sua organização, o Microsoft 365 permite que você disponibilize calendários para "usuários externos", aqueles que precisam ver o horário de disponibilidade, mas não têm contas de usuário para seu ambiente do Microsoft 365. Para obter mais informações, consulte [compartilhar calendários com usuários externos](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
