# <a name="application-registration"></a>Registro de aplicativo

1. Registre um aplicativo com o [Azure Active Directory:](https://docs.microsoft.com/powerapps/developer/data-platform/walkthrough-register-app-azure-active-directory)este tutorial descreve como registrar um aplicativo com o Azure Active Directory, que permite que um usuário com a conta de usuário do Power Apps se conecte ao ambiente do Microsoft Dataverse a partir de aplicativos cliente externos usando a autenticação OAuth.
1. [Registre](https://docs.microsoft.com/azure/active-directory/develop/quickstart-register-app)um aplicativo na Plataforma de Identidade da Microsoft: neste guia de início rápido, registre um aplicativo no portal do Azure para que a Plataforma de Identidade da Microsoft possa fornecer serviços de autenticação e autorização para seu aplicativo e seus usuários.
