# <a name="get-started-with-advanced-hunting-queries"></a>Começar com consultas de busca avançadas

Os pesquisadores de segurança da Microsoft compartilham regularmente [consultas avançadas de busca no GitHub](https://go.microsoft.com/fwlink/?linkid=2144624).

Os pesquisadores também fornecem consultas avançadas de busca para localizar atividades e indicadores associados a [ameaças emergentes.](https://go.microsoft.com/fwlink/?linkid=2145808) Essas consultas são fornecidas como parte dos relatórios de análise de ameaças, no Centro de Segurança do Microsoft Defender.
