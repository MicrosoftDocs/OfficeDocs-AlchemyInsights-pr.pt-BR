# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Usando uma caixa de correio compartilhada em um dispositivo móvel

Você não pode acessar uma caixa de correio compartilhada do aplicativo do Outlook instalado no dispositivo móvel. Quer esse recurso? Informe a equipe de engenharia do Outlook. [Vote aqui](https://go.microsoft.com/fwlink/?linked=862116)!
  
Você só pode usar uma caixa de correio compartilhada em um dispositivo móvel abrindo seu navegador, entrando no Office 365 e, em seguida, acessando o Outlook na Web. Para obter instruções, consulte [Adicionar uma caixa de correio compartilhada no Outlook na Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Se você precisar acessar caixas de correio compartilhadas usando o aplicativo Outlook em um dispositivo móvel, considere a criação de um grupo do Office 365 em vez de usar uma caixa de correio compartilhada. Para obter mais informações, consulte [Compare groups](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx).
  
Saiba mais: [obter ajuda no aplicativo para o Outlook para IOS e Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
