# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Onboard a Windows server to Microsoft Defender for Endpoint

Para o sistema operacional Windows Server, o Microsoft Defender para Ponto de Extremidade fornece recursos avançados para detectar e investigar ataques. Esses recursos são fornecidos perfeitamente, por meio do console do Centro de Segurança do Microsoft Defender.

O Defender for Endpoint dá suporte às seguintes versões do Windows Server:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) versão 1803 e posterior
- Windows Server 2019 e posterior
- Edição principal do Windows Server 2019

Para saber mais, consulte [Onboard Windows servers to the Microsoft Defender for Endpoint service](https://go.microsoft.com/fwlink/?linkid=2143627).
