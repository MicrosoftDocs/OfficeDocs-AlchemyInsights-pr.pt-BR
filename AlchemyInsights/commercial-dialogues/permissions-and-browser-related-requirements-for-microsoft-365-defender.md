# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Permissões e requisitos relacionados ao navegador para o Microsoft 365 Defender

- Permissões: para ativar o Microsoft 365 Defender, você deve ser um administrador global ou um administrador de segurança no Azure Active Directory. Para ver a lista de funções que podem usar o Microsoft 365 Defender e saber como o acesso aos dados é regulado, consulte Gerenciar o acesso ao [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626).

- Requisitos relacionados ao navegador: para acessar o Microsoft 365 Defender no centro de segurança do Microsoft 365, use o Microsoft Edge, o Internet Explorer 11 ou qualquer navegador da Web compatível com HTML 5.
