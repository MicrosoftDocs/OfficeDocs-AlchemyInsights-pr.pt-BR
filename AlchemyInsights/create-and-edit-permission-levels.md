# <a name="working-with-sharepoint-permission-levels"></a>Trabalhar com níveis de permissão do SharePoint

O SharePoint vem com níveis de permissão padrão que você pode usar, como **contribuir** e **exibir apenas**. Mas se uma delas não atender às suas necessidades, você poderá criar um novo nível de permissão. Tenha em mente que é mais fácil manter um controle dos níveis de permissão se você não alterar os padrões.

[Como criar e editar níveis de permissão](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
