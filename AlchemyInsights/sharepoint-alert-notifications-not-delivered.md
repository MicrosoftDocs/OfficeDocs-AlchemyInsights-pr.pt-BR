# <a name="sharepoint-alert-notifications-not-delivered"></a>Notificações de alerta do SharePoint não entregues 

<p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Determine se <strong>todos os alertas</strong> não são entregues ou se um <strong>alerta individual</strong> de um arquivo ou biblioteca específico não é entregue.</span></p> <ol> <li><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Se <strong style="mso-bidi-font-weight: normal;">todos os alertas</strong> de vários arquivos ou bibliotecas não forem entregues, visite o <a href="https://admin.microsoft.com/AdminPortal/Home#/servicehealth">painel de integridade do serviço</a> para verificar se há avisos/incidentes que possam ocorrer com o SharePoint ou o Exchange. O problema pode ser com o recurso de alerta do SharePoint ou atrasos em emails através do Exchange. Também será importante observar se outros emails estão sendo entregues e, caso contrário, o problema provavelmente ocorrerá com os atrasos do Exchange. <br /><br /></span></li> <li style="line-height: normal; font-size: 11pt; font-style: normal; font-weight: 400;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Se um</span> <strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">alerta individual</span></strong> a <span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">partir de um arquivo ou biblioteca específico não é entregue, você pode tentar excluir e recriar. Consulte</span> <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> <a href="https://support.office.com/en-us/article/manage-view-or-delete-sharepoint-alerts-99dfb19c-9a90-4a8c-aba1-aa8c8afb0de2#ID0EAADAAA=Online">gerenciar, exibir ou excluir alertas do SharePoint</a> para recriar o alerta.</span></li> </ol> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Para obter mais informações sobre como configurar alertas, consulte <span style="color: #333333;"> <a href="https://support.office.com/en-us/article/create-an-alert-to-get-notified-when-a-file-or-folder-changes-in-sharepoint-e5a79e7b-a146-46da-a9ef-d65409ba8918">criar um alerta para ser notificado quando um arquivo ou pasta é alterado no SharePoint</a>.</span></span></p> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Para obter mais informações sobre como solucionar problemas de alertas, consulte <a href="https://support.office.com/en-us/article/users-don-t-receive-sharepoint-online-alert-notifications-14fc22dd-e873-482c-844d-f67ad41313f1">os usuários&rsquo;que não recebem notificações de alerta do SharePoint Online</a>.</span></p> <p>&nbsp;</p>
