# <a name="error-404-when-accessing-files-in-sharepoint-or-onedrive"></a>Erro 404 ao acessar arquivos no SharePoint ou no OneDrive

O **erro 404** é recebido quando os usuários tentam acessar um site ou arquivo no SharePoint ou no onedrive. Isso geralmente é causado por um site ou arquivo ou grupo sendo renomeado, movido ou excluído.
Os usuários ocorrerão um erro 404 ao tentar acessar o conjunto de sites raiz e ele foi excluído.

O seguinte ajudará você a resolver o erro 404 para um site que foi renomeado, movido ou excluído:

- [Restaurar um conjunto de sites excluído](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection): para sites clássicos existentes no centro de administração clássico.
- [Exibir e restaurar sites excluídos no novo centro de administração do SharePoint](https://docs.microsoft.com/sharepoint/view-and-restore-deleted-sites-in-new-admin-center): para sites modernos (comunicação, grupo conectado ou outros sites) que existem na nova visualização do centro de administração.
- [Restaurar itens da lixeira de um site do SharePoint](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be): para resolver o arquivo (ou outro item) que foi renomeado, movido ou excluído, vá para o site do SharePoint ou onedrive e exiba a lixeira do conteúdo do site.
- [Pesquisar o log de auditoria no centro de conformidade &amp; de segurança do Office 365](https://support.office.com/client/search-the-audit-log-in-the-office-365-security-compliance-center-0d4d0f35-390b-4518-800e-0c7ec95e946c): Pesquisar o log de auditoria (se o registro em log estiver habilitado) se você ainda não conseguir encontrar o item.
