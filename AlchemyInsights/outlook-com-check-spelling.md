# <a name="use-your-browser-to-check-spelling"></a>Usar o navegador para verificar a ortografia

Embora o Outlook.com não tenha uma função de verificação ortográfica, vários navegadores da Web verificam automaticamente a ortografia, incluindo o Microsoft Edge, o Internet Explorer 10 e versões posteriores e as versões atuais do Firefox, do Chrome e do Safari. Verifique as opções do navegador da Web para saber mais sobre como verificar a ortografia.
