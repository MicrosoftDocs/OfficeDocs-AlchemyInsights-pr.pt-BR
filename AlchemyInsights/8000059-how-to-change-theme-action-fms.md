# <a name="how-to-change-your-theme"></a>Como alterar seu tema

Para alterar seu tema no Outlook.com ou no Outlook na web, vá para **configurações**![configurações](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) e escolha um tema de **tema**. Para ver todos os temas disponíveis, selecione **Exibir tudo** sob o **tema**.
