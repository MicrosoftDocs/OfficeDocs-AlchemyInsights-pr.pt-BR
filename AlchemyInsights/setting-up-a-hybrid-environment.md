# <a name="setting-up-a-hybrid-environment"></a>Configurando um ambiente híbrido

Para permitir que seus usuários trabalhem de forma perfeita, não importa se os dados estão armazenados em servidores locais ou no Office 365, talvez seja necessário configurar um ambiente híbrido. Para obter uma visão geral das suas opções, confira a [integração do Office 365 com ambientes locais](https://docs.microsoft.com/office365/enterprise/office-365-integration).
