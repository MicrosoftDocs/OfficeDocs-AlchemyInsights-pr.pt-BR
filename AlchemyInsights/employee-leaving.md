# <a name="have-an-employee-leaving-and-need-to-block-access-to-data-and-email"></a>Ter um funcionário deixando e precisar bloquear o acesso a dados e de email?

Para fazer isso:
  
1. Vá para [usuários ativos](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/users).
    
2. Selecione o usuário que você deseja bloquear. 
    
3. Ao lado de status de entrada, escolha **Editar**. 
    
4. Escolha **bloquear o usuário faça logon** \> **Salvar**. 
    
 **Observação**: saída para alguns aplicativos podem não ser imediatos. Acompanhar com mais ações neste artigo: [Remover um antigo funcionário do Office 365](https://support.office.com/article/https://support.office.com/en-us/article/Remove-a-former-employee-from-Office-365-44d96212-4d90-4027-9aa9-a95eddb367d1.aspx)
