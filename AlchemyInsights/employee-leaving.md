# <a name="have-an-employee-leaving-and-need-to-block-access-to-data-and-email"></a>Um funcionário sai e precisa bloquear o acesso a dados e email?
  
1. Vá para [usuários ativos](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users).

2. Selecione o usuário que você deseja bloquear.

3. Ao lado de status de entrada, escolha **Editar**.

4. Escolha **bloquear o usuário de entrar** \> para **salvar**.

 **Observação**: a saída para alguns aplicativos pode não ser imediata. Acompanhar mais ações neste artigo: [remover um antigo funcionário do Office 365](https://support.office.com/article/Remove-a-former-employee-from-Office-365-44d96212-4d90-4027-9aa9-a95eddb367d1.aspx)
