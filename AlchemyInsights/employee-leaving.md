# <a name="block-access-to-a-former-employee"></a>Bloquear o acesso a um funcionário antigo

Um funcionário sai e precisa bloquear o acesso a dados e email? Faça o seguinte.
  
1. No centro de administração, vá para a página [usuários ativos](https://go.microsoft.com/fwlink/p/?linkid=834822) do **usuário** \> .

2. Selecione o nome do usuário que você deseja bloquear.

3. Sob o nome do usuário, selecione **bloquear este usuário**.

4. \> Escolha **bloquear a assinatura do usuário** para **salvar alterações**.

**Observação**: a saída para alguns aplicativos pode não ser imediata. Acompanhar mais ações neste artigo: [remover um funcionário antigo](https://docs.microsoft.com/office365/admin/add-users/remove-former-employee)
