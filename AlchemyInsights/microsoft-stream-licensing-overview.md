# <a name="microsoft-stream-licensing-overview"></a>Visão geral do licenciamento do Microsoft Stream

O Microsoft Stream é um serviço de nuvem licenciado por assinatura por usuário. Ele está disponível como parte das assinaturas qualificadas do Office 365. Veja preços do Office 365 para obter informações sobre cada tipo de assinatura.

- [Visão geral do licenciamento do Microsoft Stream](https://docs.microsoft.com/en-us/stream/license-overview)
