# <a name="what-is-autosave"></a>O que é salvamento automático?

O salvamento automático é um recurso disponível nos assinantes do Excel, Word e PowerPoint para Microsoft 365 que salva o arquivo automaticamente, a cada alguns segundos, conforme você trabalha. 

Para perguntas frequentes, confira [o que é o salvamento automático?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
