# <a name="move-a-phone-number-from-another-provider-to-microsoft"></a>Mover um número de telefone de outro provedor para a Microsoft

Você precisa mover um número de telefone de outro provedor para a Microsoft? Números de telefone para diferentes países e regiões podem ser encontrados em [gerenciar números de telefone para sua organização](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization). Escolha seu país ou região para encontrar informações sobre a transferência de números.
