# <a name="access-services"></a>Serviços do Access

O recurso de serviços do Access no SharePoint Online está sendo desativado. Novos aplicativos web baseados em Access e bancos de dados da web não é mais podem ser criados e bancos de dados de web e de aplicativos web existentes serão desligados por abril de 2018. Bancos de dados da área de trabalho do Access (. accdb) não são afetados por essa alteração. É recomendável que powerapps Microsoft como uma alternativa para acesso web apps e bancos de dados da web. 
  
[Saiba mais sobre o Microsoft PowerApps](https://powerapps.microsoft.com/)
