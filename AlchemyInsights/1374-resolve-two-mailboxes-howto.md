# <a name="a-user-has-two-mailboxes"></a>Um usuário tem duas caixas de correio

Ambientes híbridos que usam o Azure Active Directory Connect (conexão AAD) ou DirSync acidentalmente podem causar um usuário tenha duas caixas de correio: uma na organização local e uma na nuvem.
  
Para resolver esse problema, consulte [Permanentemente limpar Info da caixa de correio anterior](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
