Configure seu aplicativo Serviço de Aplicativo ou Funções do Azure para usar o logon do [Azure AD](https://docs.microsoft.com/azure/app-service/configure-authentication-provider-aad) mostra como configurar o Serviço de Aplicativo do Azure ou as Funções do Azure para usar o Azure Active Directory (Azure AD) como um provedor de autenticação.
