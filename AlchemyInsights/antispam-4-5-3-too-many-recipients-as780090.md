# <a name="453-too-many-recipients-as780090"></a>4.5.3 excesso de destinatários (AS780090)

Esse erro ocorre quando o volume de tráfego de email do endereço IP de origem excede o limite com base na reputação (ou falta de reputação) do endereço IP de origem.

O bloqueio de email do endereço IP de origem expirará dentro de uma hora. Se o endereço IP de origem for um servidor de email local que pertença a você, verifique a configuração do conector de fluxo de emails. Se o comportamento persistir por mais de uma hora, entre em contato com o suporte para solicitar uma exceção para o endereço IP de origem.
