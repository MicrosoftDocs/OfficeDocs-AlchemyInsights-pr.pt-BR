# <a name="solutions-for-error-30180-28"></a>Soluções para o erro 30180-28

Você pode ver o [erro 30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) durante uma atualização do Office se você tiver uma conexão com a Internet ruim. Here are some things you can try to fix the problem. 
  
- ReInicie o computador e [atualize o Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA).
    
- Siga as etapas acima, mas use uma conexão com fio ao atualizar o Office.
