# <a name="change-domain-name-in-sharepoint-online"></a>Alterar o nome de domínio no SharePoint Online

Infelizmente, não é possível alterar o nome de domínio do SharePoint Online para sua organização no Office 365. Por exemplo, se o nome da sua organização mudar de "contoso" para "Fourth Coffee", não será possível alterar o contoso.sharepoint.com para o fourthcoffee.sharepoint.com.
  
Para usar o nome de domínio fourthcoffee.sharepoint.com, você precisará comprar uma nova assinatura do Office 365 e mover todos os emails, arquivos e outros dados que você deseja manter para a nova assinatura.
  
 **Observação**: por exemplo, para alterar o nome de um site de https://contoso.sharepoint.com/sites/sample1 para https://contoso.sharepoint.com/sites/sample2, crie um novo site e mova todo o conteúdo do site antigo para o novo.
  
Para obter informações sobre como mover itens em bibliotecas de documentos, consulte [mover ou copiar itens de uma biblioteca de documentos do SharePoint](https://go.microsoft.com/fwlink/?linkid=2025831).
