# <a name="change-domain-name-in-sharepoint-online"></a>Alterar o nome de domínio no SharePoint Online

Infelizmente, não é possível alterar o nome de domínio do SharePoint Online para sua organização no Office 365. Por exemplo, se o nome da sua organização for alterada de "Contoso" para "Fourth Coffee", você não pode alterar contoso.sharepoint.com para fourthcoffee.sharepoint.com.
  
Para usar o fourthcoffee.sharepoint.com de nome de domínio, você precisaria adquirir uma nova assinatura do Office 365 e mover todos os emails, arquivos e quaisquer outros dados que você deseja manter para a nova assinatura.
  
 **Observação**: por exemplo, para alterar o nome de um site de https://contoso.sharepoint.com/sites/sample1 para https://contoso.sharepoint.com/sites/sample2, crie um novo site e, em seguida, mover todo o conteúdo do site antigo para o novo. 
  
Para informações sobre como mover itens em bibliotecas de documentos, consulte [Mover ou copiar itens em uma biblioteca de documentos do SharePoint](https://go.microsoft.com/fwlink/?linkid=2025831).
