# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>"Usuário desconhecido" aparece nas reuniões ou chats do Microsoft Teams

O usuário pode ter sido excluído do Microsoft 365. Nesse caso, tente excluir o usuário do teams.  

Tente limpar o seguinte cache quando o Microsoft Teams estiver fechado e reinicie o Teams: `C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

Para obter atualizações sobre esse problema, confira o  [usuário removido aparece como "usuário desconhecido" no Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown).
