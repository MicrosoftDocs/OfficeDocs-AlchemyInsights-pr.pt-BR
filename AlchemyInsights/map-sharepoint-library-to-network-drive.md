# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Mapear uma biblioteca do SharePoint para uma unidade de rede

Para trabalhar com bibliotecas de site no Gerenciador de arquivos, é recomendável usar o cliente de sincronização do OneDrive em vez de mapeamento de bibliotecas em unidades de rede. Mapeamento de unidade é apenas uma solução temporária. Usar o cliente de sincronização disponibiliza os arquivos para usuários mesmo quando eles estão offline. O cliente de sincronização também fornece um melhor desempenho, permite acesso aos arquivos sem precisar baixá-los usando [Arquivos sob demanda](https://support.office.com/en-us/article/Learn-about-OneDrive-Files-On-Demand-0E6860D3-D9F3-4971-B321-7092438FB38E) e não exige a instalação usando o Internet Explorer. 
  
- [Sincronizar arquivos com o novo cliente de sincronização do OneDrive no Windows](https://go.microsoft.com/fwlink/?linkid=866427).
    
- [Maneiras de trabalhar com arquivos de biblioteca do site no Explorador de Arquivos](https://go.microsoft.com/fwlink/?linkid=866291)
    
- [Configurar e solucionar problemas de unidades de rede mapeadas](https://support.microsoft.com/kb/2616712)
