# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Converter uma caixa de correio do usuário em uma caixa de correio compartilhada

Quando você converte caixas de correio de um usuário em uma caixa de correio compartilhada, todos os email existente e calendário é mantida. Somente agora está em uma caixa de correio compartilhada onde várias pessoas poderão acessá-la, em vez de uma pessoa. Posteriormente, você pode converter uma caixa de correio compartilhada volta para uma caixa de correio do usuário (privada).
