# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Converter uma caixa de correio de usuário em uma caixa de correio compartilhada

Quando você converte a caixa de correio do Office 365 de um usuário individual em uma caixa de correio compartilhada, todos os itens de email e calendário existentes são mantidos em uma caixa de correio compartilhada onde várias pessoas podem acessar. Você também pode converter uma caixa de correio compartilhada de volta para uma caixa de correio de usuário (privada).
  
Para obter mais informações, consulte [converter uma caixa de correio de usuário em uma caixa de correio compartilhada](https://docs.microsoft.com/office365/admin/email/convert-user-mailbox-to-shared-mailbox).
