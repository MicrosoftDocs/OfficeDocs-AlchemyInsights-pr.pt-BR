# <a name="cutover-migrations-to-office-365"></a>Migrações de substituição para o Office 365

Ao se inscrever no Office 365, você pode migrar o conteúdo das caixas de correio de usuário de uma organização local do Exchange para o Office 365. Quando você faz isso tudo de uma só vez, ele é chamado de migração de substituição. Uma migração de substituição pode ser adequada se você estiver executando uma versão compatível do Exchange e tiver menos de 2.000 caixas de correio. Para saber mais sobre as migrações de substituição, confira: [migração de substituição para o Office 365](https://support.office.com/article/9496e93c-1e59-41a8-9bb3-6e8df0cd81b4.aspx).
