# <a name="cutover-migrations-to-office-365"></a>Migrações de transferência para o Office 365

Quando você se inscreve para o Office 365, você pode migrar o conteúdo de caixas de correio de usuário de uma organização do Exchange no local para o Office 365. Quando você fizer isso tudo de uma só vez, ele se chama uma migração gradual. Uma migração gradual poderia ser certa para você, se você estiver executando uma versão compatível do Exchange e tem menos de 2.000 caixas de correio. Para saber mais sobre migrações de transferência, dê uma olhada: [migração para o Office 365](https://support.office.com/article/9496e93c-1e59-41a8-9bb3-6e8df0cd81b4.aspx).
