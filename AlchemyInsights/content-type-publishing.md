# <a name="content-type-publishing"></a>Publicação de tipo de conteúdo

Se você quiser que os tipos de conteúdo estejam disponíveis em sua organização para todos os usuários, é uma boa candidata a publicar em todos os conjuntos de sites. Para publicar um tipo de conteúdo, confira [publicar um tipo de conteúdo de um hub de publicação de conteúdo](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).
