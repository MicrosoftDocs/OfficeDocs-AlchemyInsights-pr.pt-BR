# <a name="restore-deleted-files-or-folders-in-sharepoint"></a>Restaurar arquivos ou pastas excluídos no SharePoint

Se você excluir acidentalmente um arquivo ou uma pasta no SharePoint, você poderá recuperá-lo mais tarde da lixeira.

* [Restaurar arquivos ou pastas excluídos no SharePoint](https://support.office.com/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b)

Se muitos de seus arquivos do SharePoint forem excluídos, substituídos, corrompidos ou infectados por malware, você poderá restaurar toda a sua biblioteca do SharePoint para uma hora anterior.

* [Restaurar uma biblioteca de documentos](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
