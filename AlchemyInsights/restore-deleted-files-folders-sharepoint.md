# <a name="restore-deleted-files-or-folders-in-sharepoint"></a>Restaurar arquivos ou pastas excluídos no SharePoint

Se você excluir acidentalmente um arquivo ou pasta no SharePoint, poderá recuperá-lo posteriormente da lixeira.

* [Restaurar arquivos ou pastas excluídos no SharePoint](https://support.microsoft.com/office/restore-items-in-the-recycle-bin-that-were-deleted-from-sharepoint-or-teams-6df466b6-55f2-4898-8d6e-c0dff851a0be)

Se muitos dos seus arquivos do SharePoint são excluídos, substituídos, corrompidos ou infectados por malware, você pode restaurar toda a biblioteca do SharePoint para uma hora anterior.

* [Restaurar uma biblioteca de documentos](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
