# <a name="microsoft-graph-rest-api-interface"></a>Interface de API REST do Microsoft Graph

Os conjuntos de APIs do Graph no ponto de extremidade do v 1.0 estão no status de disponibilidade geral (GA) e passam por um processo rigoroso de análise e feedback com os clientes para atender às necessidades práticas e de produção.

- Para obter mais informações sobre a API do Graph v 1.0, consulte [referência da API REST do Microsoft Graph v 1.0](https://docs.microsoft.com/graph/api/overview?toc=.%2Fref%2Ftoc.json&view=graph-rest-1.0). 
- Para obter mais informações sobre a versão beta da API do Graph, consulte [referência de ponto de extremidade beta do Microsoft Graph](https://docs.microsoft.com/graph/api/overview?toc=.%2Fref%2Ftoc.json&view=graph-rest-beta).

Para obter mais informações sobre o Microsoft Graph, consulte a [documentação do Microsoft Graph](https://docs.microsoft.com/graph/).
