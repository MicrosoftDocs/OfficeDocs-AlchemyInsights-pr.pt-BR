# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Recebendo um erro de permissão ao abrir uma caixa de correio compartilhada no Outlook na web?

Se você receber o erro **algo deu errado. Você não tem permissão para abrir esta caixa de correio.** Quando você tenta abrir uma caixa de correio compartilhada no Outlook na web, sua conta talvez não tenha permissão para acessar a caixa de correio compartilhada. O administrador da sua organização pode atribuir das permissões da conta na caixa de correio compartilhada seguindo as instruções no tópico [criar uma caixa de correio compartilhada](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).
