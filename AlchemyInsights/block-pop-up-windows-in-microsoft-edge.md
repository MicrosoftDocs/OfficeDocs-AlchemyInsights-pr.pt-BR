# <a name="block-pop-up-windows-in-microsoft-edge"></a>Bloquear janelas pop-up no Microsoft Edge

1. Vá para **... (Configurações e muito mais)**  >  **Configurações**  >  **Permissões de site**.
2. Selecione **Pop-ups e redirecionamentos.**
3. Ativar **Bloqueio**.
