# <a name="sharepoint-online-throttling"></a>Otimização do SharePoint Online

SharePoint Online usa a limitação para manter o melhor desempenho e confiabilidade do serviço SharePoint Online. A limitação limita o número de ações do usuário ou de chamadas simultâneas (por script ou código) a fim de impedir o uso excessivo de recursos. Se você obter limitadas, 99% do tempo é por causa de código personalizado.
  
Para obter mais informações sobre a limitação, confira o artigo sobre [como ficar limitado ou bloqueado no SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
