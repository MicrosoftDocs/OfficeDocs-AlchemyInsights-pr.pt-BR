# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Use SaRA para diagnosticar e resolver problemas de autenticação do Outlook

É altamente recomendável que você usa o [Outlook continua solicitando minha senha](https://aka.ms/SaRA-OutlookPwdPrompt) diagnóstico na máquina afetada para solucionar problemas em que o Outlook continuamente solicitará uma senha. Este [SaRA](https://diagnostics.office.com/#/) diagnóstico não verificações automatizadas e retorna as soluções possíveis para ser usada para resolver qualquer problema detectado.
