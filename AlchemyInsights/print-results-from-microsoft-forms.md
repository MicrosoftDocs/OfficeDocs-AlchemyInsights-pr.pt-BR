# <a name="print-results-in-a-summary-of-responses"></a>Imprimir resultados em um resumo de respostas

Você pode [imprimir os resultados do Microsoft Forms](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) como um resumo das respostas ou para uma pessoa. 

Não é possível imprimir formulários sem respostas, mas você pode solicitar ou votar nesse recurso visitando o [site do Microsoft Forms UserVoice](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
