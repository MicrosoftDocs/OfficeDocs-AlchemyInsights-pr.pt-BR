# <a name="some-users-may-experience-delays-with-chat-messages"></a>Alguns usuários podem ter atrasos com mensagens de chat

A Microsoft está ciente de um problema em que os usuários podem ter atrasos no envio e recebimento de mensagens de chat ao usar o Teams.

Para obter as informações mais recentes, consulte [https://admin.microsoft.com/AdminPortal/Home#/servicehealth/advisories/:/alerts/TM240228](https://admin.microsoft.com/AdminPortal/Home#/servicehealth/advisories/:/alerts/TM240228)
