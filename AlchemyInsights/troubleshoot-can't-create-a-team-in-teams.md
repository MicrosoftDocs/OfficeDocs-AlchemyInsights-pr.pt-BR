# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>O que fazer se você não puder criar uma equipe no Microsoft Teams

Peça ao administrador do Microsoft Teams para fazer o seguinte:

1. Criar ou alterar as configurações do grupo de segurança da organização no Microsoft 365.
2. Adicione você ao grupo de segurança.
