# <a name="creating-a-team-from-an-existing-team"></a>Criar uma equipe a partir de uma equipe existente

Alguns clientes relataram um problema com a criação de uma equipe a partir de uma equipe existente (ou clonagem de uma equipe) quando optaram por copiar os membros da equipe. Para resolver esse problema, não copie os membros como parte do processo de criação e, em vez disso, adicione membros após a criação da equipe.

Para obter mais informações, [consulte Criar uma equipe de uma equipe existente.](https://support.microsoft.com/office/create-a-team-from-an-existing-team-f41a759b-3101-4af6-93bd-6aba0e5d7635)
