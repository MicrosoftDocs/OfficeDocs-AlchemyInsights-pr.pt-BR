# <a name="get-the-a1-plan"></a>Obter o plano A1

Se o seu Instituto já estiver qualificado para o Microsoft 365 Education, você pode adicionar a assinatura A1 do **Centro de administração do Microsoft 365**. 

1. Vá para **Faturamento > [Produtos e serviços](https://go.microsoft.com/fwlink/p/?linkid=868433)**.

2. Use a caixa de pesquisa para Pesquisar "A1".

3. Selecione o número de usuários que usarão o plano A1.

Se o seu Instituto ainda não configurou o Microsoft 365 Education, mas você está tentando adicionar o Office 365 gratuitamente, [comece a usar ](https://www.microsoft.com/education/products/office) fornecendo o seu e-mail.
