# <a name="microsoft-365-language-settings"></a>Configurações de idioma do Microsoft 365

Determine se você deseja alterar o idioma de exibição e outras configurações para usuários ou para o locatário inteiro.

- Para usuários, confira [alterar o idioma de exibição e o fuso horário no Microsoft 365 for Business para todos os aplicativos](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b).
- Para o locatário, consulte [definir as configurações de idioma e regionais para o Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region).
