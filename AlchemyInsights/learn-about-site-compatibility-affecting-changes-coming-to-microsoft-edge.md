# <a name="learn-about-site-compatibility-affecting-changes-coming-to-microsoft-edge"></a>Saiba mais sobre as alterações que afetam a compatibilidade do site no Microsoft Edge

A Web está em constante evolução para melhorar a experiência do usuário, a segurança e a privacidade. Algumas alterações são significativas o suficiente para afetar a funcionalidade das páginas existentes. A equipe do Microsoft Edge atualiza as páginas à medida que o pensamento evolui, as linhas do tempo se solidificam e [as alterações são anunciadas](https://go.microsoft.com/fwlink/?linkid=2135534).
