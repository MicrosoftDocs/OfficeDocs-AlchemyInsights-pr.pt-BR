# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Obtendo uma conta desativada erro no Outlook na web?

Se você receber o erro que **sua conta tiver sido desativada** com `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` ao tentar abrir o Outlook na web (anteriormente conhecido como o Outlook Web App), administração de sua organização poderá ter desabilitado o seu acesso ao Outlook na web. O administrador pode corrigir este erro seguindo as etapas no tópico [Habilitar ou desabilitar o Outlook Web App para uma caixa de correio](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
