<span data-ttu-id="e38c0-p101">Se você receber o erro que **sua conta tiver sido desativada** com `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` ao tentar abrir o Outlook na web (anteriormente conhecido como o Outlook Web App), administração de sua organização poderá ter desabilitado o seu acesso ao Outlook na web. O administrador pode corrigir este erro seguindo as etapas no tópico [Habilitar ou desabilitar o Outlook Web App para uma caixa de correio](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).</span><span class="sxs-lookup"><span data-stu-id="e38c0-p101">If you get the error **Your account has been disabled** with  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` when you try to open Outlook on the web (formerly known as Outlook Web App), your organization admin might have disabled your access to Outlook on the web. Your admin can fix this error by following the steps in the topic [Enable or disable Outlook Web App for a mailbox](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).</span></span>
