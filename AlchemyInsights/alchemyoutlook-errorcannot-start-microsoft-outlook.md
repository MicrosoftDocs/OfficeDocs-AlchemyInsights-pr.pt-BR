# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Erro do Outlook: "Não é possível iniciar o Microsoft Outlook"

Você está tendo problemas para iniciar o Outlook 2016, Outlook 2010 ou Outlook 2013 ou recebendo "não é possível iniciar o Microsoft Outlook. Não é possível abrir a janela do Outlook"mensagens de erro?
  
Primeiro, use as etapas de solução de problemas [neste](https://support.office.com/en-us/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) artigo para ajudar a resolver esses problemas de inicialização. 
  
Também é possível executar a [ferramenta de diagnóstico avançado](https://aka.ms/SaRA-OutlookAdvDiagnostics) na máquina afetada.
