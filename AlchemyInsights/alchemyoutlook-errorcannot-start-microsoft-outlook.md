# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Erro do Outlook: "não é possível iniciar o Microsoft Outlook"

Você está tendo problemas para iniciar o Outlook 2016, o Outlook 2013 ou o Outlook 2010 ou o recebimento de "não é possível iniciar o Microsoft Outlook. Não é possível abrir as mensagens de erro da janela do Outlook?
  
Primeiro, use as etapas de solução de problemas neste [artigo](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) para ajudar a resolver esses problemas de inicialização. 
  
Você também pode executar a [ferramenta de diagnóstico avançado](https://aka.ms/SaRA-OutlookAdvDiagnostics) na máquina afetada.
