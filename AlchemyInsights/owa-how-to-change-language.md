# <a name="change-language"></a>Alterar Idioma

1.    Entre no Outlook na Web.
2. Na parte superior da página, selecione **configurações** > **Exibir todas as configurações do Outlook**.
3. Selecione idioma **geral** > **e hora**.
4. Selecione o idioma que você deseja usar.
5. Selecione **Salvar**.
 
Para saber mais, confira [alterar as configurações de fuso horário e idioma no Outlook na Web](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
