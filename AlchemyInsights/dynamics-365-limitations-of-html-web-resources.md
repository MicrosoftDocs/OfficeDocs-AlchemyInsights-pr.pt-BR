# <a name="webpage-html-web-resources-limitations"></a>Limitações de recursos da Web da página da Web (HTML)

* Um recurso Web HTML não pode conter qualquer código que deva ser executado no servidor. As páginas do ASP.NET não podem ser carregadas como recursos HTML da Web.

* Os recursos da Web HTML só podem aceitar um número limitado de parâmetros de cadeia de caracteres de consulta. Mais informações: [passar parâmetros para recursos HTML da Web](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
