# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Converter uma caixa de correio do usuário em uma caixa de correio de sala ou equipamento

Você também pode converter uma caixa de correio de tipo de uma caixa de correio para outro, mas você precisará executar essas tarefas usando o PowerShell do Exchange Online. Confira [converter uma caixa de correio](https://go.microsoft.com/fwlink/p/?LinkId=832875) para obter mais detalhes.
