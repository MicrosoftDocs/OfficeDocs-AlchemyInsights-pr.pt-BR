# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Encontre resultados na Pesquisa do Bing usando aplicativos lógicos do Azure

Usando a Pesquisa do Bing de dentro de um aplicativo lógico, você pode automatizar o processamento dos resultados da pesquisa e disponibilizar esses resultados para outras ações. Por exemplo, você pode encontrar itens de notícias e fazer com que o Twitter os poste como tweets no feed.

Para saber mais, confira [Encontrar resultados na Pesquisa do Bing usando aplicativos lógicos do Azure.](https://go.microsoft.com/fwlink/?linkid=2151928)
