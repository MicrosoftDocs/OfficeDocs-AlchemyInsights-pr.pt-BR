# <a name="delete-a-user-and-remove-licenses"></a>Excluir um usuário e remover licenças

Excluir um usuário também cancelará a atribuição das licenças de usuário, disponibilizando-os para outros usuários. 
1. No centro de administração, vá para **usuários** > usuários ativos.
1. Selecione o usuário ou os usuários a serem excluídos e clique em **excluir usuário**.

Para obter mais informações, consulte [excluir um usuário da sua organização](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user).
