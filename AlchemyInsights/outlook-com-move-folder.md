# <a name="move-a-folder"></a>Mover uma pasta

Você pode arrastar pastas em **favoritos** para reordenar.
  
Você não pode reordenar pastas em **pastas** ou mover uma subpasta para se tornar uma pasta.

*Observação:* Se você mover uma pasta para se tornar uma subpasta, crie uma nova pasta e copie todos os seus emails da subpasta para a nova pasta.
  
Para saber mais, confira [trabalhar com pastas de mensagens no Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
