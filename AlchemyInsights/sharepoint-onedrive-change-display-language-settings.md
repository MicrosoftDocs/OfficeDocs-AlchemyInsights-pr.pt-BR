Para obter mais informações sobre as configurações de idioma de <a title="solução de problemas, consulte não é possível alterar o idioma padrão de um conjunto de sites do SharePoint Online" href="https://support.office.com/en-us/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964">Não é possível alterar o idioma padrão de um conjunto</a>de sites do SharePoint Online.</li> </ol> <p>&nbsp;</p>
