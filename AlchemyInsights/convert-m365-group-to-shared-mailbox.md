# <a name="conversion-of-microsoft-365-group-and-shared-mailboxes"></a>Conversão de grupos do Microsoft 365 e caixas de correio compartilhadas

No momento, não há uma maneira de converter entre a caixa de correio de grupo do Microsoft 365 e uma caixa de correio compartilhada.

Para obter este recurso, informe [aqui](https://aka.ms/M365GroupToShared).
