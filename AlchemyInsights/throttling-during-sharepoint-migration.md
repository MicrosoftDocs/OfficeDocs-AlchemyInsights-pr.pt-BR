# <a name="sharepoint-migration-throttling"></a>Otimização da migração do SharePoint

SharePoint Online usa a limitação para manter o melhor desempenho e confiabilidade do serviço SharePoint Online. A limitação limita o número de ações do usuário ou de chamadas simultâneas (por script ou código) a fim de impedir o uso excessivo de recursos. Para obter mais informações, consulte:

- [Estou com desempenho fraco ou limitação durante a migração](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
