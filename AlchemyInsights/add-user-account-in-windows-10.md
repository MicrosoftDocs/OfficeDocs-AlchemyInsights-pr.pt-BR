# <a name="add-a-user-account-in-windows-10"></a>Adicionar uma conta de usuário no Windows 10

Veja como adicionar uma conta no Windows 10:

1. Vá para **Iniciar** Configurações Família de Contas & outras pessoas (ou Outras pessoas , se você estiver usando  >    >    >   o Windows 10 Enterprise) > Adicionar  outra pessoa a este computador .
2. Siga as instruções para adicionar uma conta da Microsoft ou conta local.

[Saiba mais sobre como configurar contas](https://support.microsoft.com/help/17197/)
