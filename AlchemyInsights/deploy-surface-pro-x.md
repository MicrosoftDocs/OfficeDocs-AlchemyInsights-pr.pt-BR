# <a name="deploy-surface-pro-x"></a>Implantar Surface Pro X

Usar o Windows Autopilot para implantar dispositivos Surface X Pro para sua organização oferece os benefícios de uma implantação sem toque baseada em nuvem com o Office Pro Plus. Para saber mais sobre o Autopilot, confira [Visão geral do Windows Autopilot Deployment](https://docs.microsoft.com/mem/autopilot/windows-autopilot).

**Observação:** as implantações tradicionais baseadas em imagem não estão disponíveis para dispositivos Surface Pro X.

As etapas exatas para implantar dispositivos Surface usando o Autopilot variam de acordo com a situação. Para encontrar as melhores etapas para o seu dispositivo, confira [Cenários e recursos do Windows Autopilot Deployment](https://docs.microsoft.com/mem/autopilot/windows-autopilot-scenarios).
