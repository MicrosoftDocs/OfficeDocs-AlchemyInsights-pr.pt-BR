# <a name="import-contacts"></a>Importar contatos

Antes de importar contatos para o Outlook na Web, você precisará exportá-los do seu outro provedor de email como um arquivo. csv. Em seguida, siga as instruções abaixo.

1. Na parte inferior da página, selecione o ícone pessoas para ir para a página pessoas.
2. Na barra de ferramentas, selecione **gerenciar** > **contatos de importação**.
3. Selecione **procurar**, escolha o arquivo CSV e, em seguida, selecione **abrir**.
4. Selecione **Importar**.

Para saber mais, confira [importar contatos para o Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
