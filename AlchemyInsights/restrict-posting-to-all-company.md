# <a name="restrict-posting-to-all-company"></a>Restringir a postagem em Todas as Empresas

É possível restringir [todas as postagens da Empresa no Yammer](https://support.microsoft.com/office/restrict-all-company-posts-in-yammer-3219d2ae-db15-4c9f-9dd2-28559ae39a97) usando uma configuração disponível para administradores. Os usuários ainda poderão responder aos threads iniciados pelos administradores, a menos que os administradores fechem explicitamente [as conversas.](https://support.microsoft.com/office/pin-close-and-report-conversations-in-yammer-62a5fbc2-ff1b-4418-9334-d2b4b17062cb) Não é possível restringir a postagem em outras comunidades do Yammer no momento.
