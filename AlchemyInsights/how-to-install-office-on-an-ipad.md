# <a name="how-to-install-office-on-an-ipad"></a>Como instalar o Office em um iPad

- Peça para o usuário [tocar no link de download a](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA) partir de seu dispositivo móvel, para Word, Excel, PowerPoint e OneNote.

- Faça com que o usuário inicie o aplicativo, toque **em entrar** e insira suas credenciais corporativas ou de estudante.
