# <a name="use-your-browser-to-check-spelling"></a>Usar o navegador para verificar a ortografia

Não há opção de verificação ortográfica no Outlook na Web. Para verificar a ortografia, você precisará usar o navegador da Web. A verificação ortográfica está disponível no Microsoft Edge, no Internet Explorer 10 e nas versões posteriores e nas versões atuais do Firefox, do Chrome e do Safari. Verifique as opções do navegador da Web para saber mais sobre como verificar a ortografia.
