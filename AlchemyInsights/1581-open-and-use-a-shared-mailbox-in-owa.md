# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>Abrir e usar uma caixa de correio compartilhada no Outlook na web

Compartilhadas caixas de correio que você tem acesso ao automaticamente não são exibidas no Outlook na web (anteriormente conhecido como o Outlook Web App).

Você pode exibir a caixa de correio compartilhada e sua caixa de correio principal juntos ou abrir a caixa de correio compartilhada em uma janela separada do navegador. Para receber notificações de novas mensagens de caixa de correio compartilhada, você precisará abrir a caixa de correio compartilhada em uma janela separada do navegador. Para obter mais informações, consulte [Abrir e usar uma caixa de correio compartilhada no Outlook Web App](https://support.office.com/article/BC127866-42BE-4DE7-92AE-1EF2F787FD5C).
