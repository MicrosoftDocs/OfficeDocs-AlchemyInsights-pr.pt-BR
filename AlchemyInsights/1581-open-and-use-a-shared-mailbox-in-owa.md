# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>Abrir e usar uma caixa de correio compartilhada no Outlook na Web

Caixas de correio comPartilhadas às quais você tem acesso não são automaticamente exibidas no Outlook na Web (anteriormente conhecido como Outlook Web App).

Você pode exibir a caixa de correio compartilhada e a caixa de correio principal juntas ou abrir a caixa de correio compartilhada em uma janela separada do navegador. Para receber novas notificações de mensagem para a caixa de correio compartilhada, você precisa abrir a caixa de correio compartilhada em uma janela separada do navegador. Para obter mais informações, consulte [abrir e usar uma caixa de correio compartilhada no Outlook Web App](https://support.office.com/article/BC127866-42BE-4DE7-92AE-1EF2F787FD5C).
