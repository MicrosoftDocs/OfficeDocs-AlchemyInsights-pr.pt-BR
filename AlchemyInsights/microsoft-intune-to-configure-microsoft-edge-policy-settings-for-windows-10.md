# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>Usar o Microsoft Intune para configurar as configurações de política do Microsoft Edge para o Windows 10

Você pode configurar políticas e configurações do Microsoft Edge adicionando um perfil de configuração de dispositivo ao Microsoft Intune. Isso equivale a configurar as políticas de grupo do Microsoft Azure Active Directory ou as configurações locais do Objeto de Política de Grupo (GPO) em dispositivos de usuário.

Para saber mais, confira [Criar um perfil para gerenciar configurações no Microsoft Edge para Windows 10](https://go.microsoft.com/fwlink/?linkid=2133700).

Este artigo se aplica às versões 77 e posteriores do Microsoft Edge.
