# <a name="profile-property-synchronization"></a>Sincronização de propriedades de perfil

Não é possível mapear propriedades adicionais para a importação do AD a ser sincronizada do Azure Active Directory para o aplicativo de perfil de usuário. A importação do AD é limitada a um conjunto pré-configurado de propriedades para garantir um desempenho consistente do trabalho de timer.
  
[Saiba mais sobre as propriedades de perfil que sincronizam](https://go.microsoft.com/fwlink/?linkid=875671)
