# <a name="profile-property-synchronization"></a>Sincronização de perfil de propriedade

Não é possível mapear propriedades adicionais para a importação do AD para a sincronização do Azure Active Directory com o aplicativo de perfil de usuário. Importação do AD está limitada a um conjunto predefinido de propriedades para garantir o desempenho consistente do trabalho de timer.
  
[Saiba mais sobre as propriedades de perfil que a sincronização](https://go.microsoft.com/fwlink/?linkid=875671)
