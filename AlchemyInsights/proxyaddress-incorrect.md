# <a name="proxyaddress-incorrect"></a>ProxyAddress incorreto

Quando um objeto é sincronizado com o Azure AD, os valores especificados no atributo proxyAddresses no Active Directory são comparados às regras do Azure AD e o atributo proxyAddresses é preenchido no Azure AD. Portanto, os valores do atributo proxyAddresses para o objeto no Active Directory podem não ser iguais aos valores do atributo proxyAddresses no Azure AD.
  
Para saber mais sobre como o proxyAddress está preenchido, confira [como o atributo ProxyAddress é preenchido no Azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
