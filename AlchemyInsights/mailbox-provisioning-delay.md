# <a name="your-users-mailbox-is-being-created"></a>A caixa de correio do usuário está sendo criada

Pode levar até uma hora para que uma nova caixa de correio esteja pronta para uso.
  
Se a caixa de correio de um usuário estiver "se preparando" por mais de uma hora, você pode tentar excluir e recriar o usuário ou usar a opção "Deixe-nos chamar" para falar com um engenheiro de suporte.
