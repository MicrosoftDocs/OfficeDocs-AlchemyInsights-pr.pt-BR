# <a name="learn-about-accessibility-in-microsoft-edge"></a>Saiba mais sobre acessibilidade no Microsoft Edge

No Microsoft Edge, introduzimos uma moderna [API de Automação de interface do usuário (API UIA)](https://go.microsoft.com/fwlink/?linkid=2153423). A mudança para o UIA é um grande investimento na acessibilidade do navegador e estabelece a base para uma experiência da web mais inclusiva para usuários que dependem de tecnologia assistiva no Windows 10. 

Para saber mais, confira [Acessibilidade no Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2153512).
