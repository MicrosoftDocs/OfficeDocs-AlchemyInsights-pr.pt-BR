# <a name="error-onedrive-cannot-connect-to-windows"></a>Erro: o OneDrive não pode se conectar ao Windows

Para resolver a mensagem de erro, procure o arquivo cldflt. sys para ver se ele está em execução e, se não estiver, habilite-o. 
  
Para saber mais sobre como habilitar o arquivo cldflt. sys, confira o [onedrive não pode se conectar ao Windows "ao acessar arquivos no onedrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
