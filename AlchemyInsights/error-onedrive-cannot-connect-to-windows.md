# <a name="error-onedrive-cannot-connect-to-windows"></a>Erro: OneDrive não consegue se conectar ao Windows

Para resolver a mensagem de erro, procure o arquivo cldflt.sys ver se ele estiver em execução e, se ainda não habilitá-lo. 
  
Para obter informações adicionais sobre como habilitar o arquivo cldflt.sys, consulte [OneDrive não consegue se conectar ao Windows "Erro ao acessar os arquivos no OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
