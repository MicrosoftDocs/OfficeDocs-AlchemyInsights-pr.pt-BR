# <a name="recommended-steps-to-take-if-an-account-is-compromised"></a>Etapas recomendadas a serem executadas se uma conta for comprometida

[VÍDEO: corrigindo uma conta do Office 365 comprometida](https://www.microsoft.com/videoplayer/embed/RE2jvOb?pid=ocpVideo0-innerdiv-oneplayer&amp;postJsllMsg=true&amp;maskLevel=20&amp;autoplay=true)
  
1. [Redefina a senha do usuário](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) imediatamente. Não comunique a nova senha por email para o usuário final. 
    
2. Remova qualquer [endereço](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) de encaminhamento suspeito definido no nível da caixa de correio. 
    
3. Remova todas [as regras de caixa de entrada](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) suspeitas definidas na caixa de correio. 
    
4. Se o usuário estiver impedido de enviar emails, [vá para os usuários restritos para desbloquear a conta](https://protection.office.com/?hash=/restrictedusers). Depois de concluído, o usuário deve ser capaz de retomar o envio de mensagens dentro de 1 hora.
    
5. Remova a conta de usuário de todos os [grupos de função administrativa](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) até ter certeza de que a conta não está mais comprometida. 
    
Para minimizar o potencial de uma violação de dados ou uma conta comprometida no futuro, recomendamos a leitura do nosso [artigo de práticas recomendadas de segurança do Office 365](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).
