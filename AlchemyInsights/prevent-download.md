# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Impedir o download de arquivos do SharePoint ou do OneDrive

Há vários métodos que podem ser usados para impedir ou bloquear downloads. Visite os links abaixo para obter os respectivos métodos:

- [Bloquear download de arquivo com o acesso condicional do Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Controlar o acesso de dispositivos não gerenciados](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Configurar políticas de IRM em bibliotecas de documentos do SharePoint para limitar o download de conteúdo](https://docs.microsoft.com/microsoft-365/compliance/set-up-irm-in-sp-admin-center)
