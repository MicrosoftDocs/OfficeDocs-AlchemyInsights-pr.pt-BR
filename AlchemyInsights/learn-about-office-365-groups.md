# <a name="learn-about-microsoft-365-groups"></a>Saiba mais sobre os grupos do Microsoft 365

Microsoft 365 groups é um serviço que funciona com as ferramentas do Office 365 que você usa, para que você possa colaborar com seus colegas de equipe ao escrever documentos, criar planilhas, trabalhar em planos de projeto, agendar reuniões ou enviar emails. Para obter mais informações, consulte [saiba mais sobre o Microsoft 365 groups](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
