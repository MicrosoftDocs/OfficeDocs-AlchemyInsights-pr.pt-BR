# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>Sincronizar arquivos do SharePoint usando o novo cliente de sincronização do OneDrive

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">O comando Abrir com o Explorer abre uma instância local do Windows Explorer que exibe a estrutura de pastas no servidor que hospeda o site do SharePoint. Dito isto, recomendamos que você <a href="https://support.office.com/pt-BR/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">sincronize os arquivos do SharePoint com o novo cliente de sincronização do OneDrive</a>, que fornece o <a href="https://support.office.com/pt-BR/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">Arquivos Sob Demanda</a>, porque ele fornece acesso local aos seus arquivos e oferece o melhor desempenho</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Se você optou por usar a Exibição do Explorer em vez do novo cliente de sincronização, siga as etapas e as práticas recomendadas nos artigos abaixo.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/pt-BR/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">Como usar o comando "Abrir com o Explorer" para solucionar problemas no SharePoint Online</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/pt-BR/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">Copie ou mova arquivos de biblioteca usando Abrir com o Explorer</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Observação:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> O botão Abrir com o Explorer não aparece na nova experiência da biblioteca. Clique no menu suspenso <strong>Exibir</strong> no canto superior direito (o nome do menu suspenso varia, dependendo da sua exibição atual), e clique em <strong>Exibir</strong> no Gerenciador de Arquivos.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Observação:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> o recurso Abrir com o Explorer do SharePoint usa <strong>controles ActiveX</strong>, portanto, só é compatível com o Internet Explorer 10 ou 11. Abrir com o Explorer não funciona no Windows com Microsoft Edge, Google Chrome, Mozilla Firefox ou na plataforma Mac. Devido a isso, a opção &ldquo;Exibição do Explorer&rdquo; pode estar desativada.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/pt-BR/article/Why-SharePoint-ribbon-buttons-are-unavailable-48b0939a-2efb-4e79-b5e8-b2c4cb5d04ca">Por que os botões da faixa de opções do SharePoint não estão disponíveis ou estão esmaecidos.</a></span></li> </ul> <p>&nbsp;</p>
