# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versão suportada da ferramenta de migração do SharePoint

No momento a ferramenta de migração do SharePoint só oferece suporte a migrações do SharePoint 2013. Não há suporte para versões 2010 e 2016 neste momento.
  
Para obter mais informações, consulte [apresentando a ferramenta de migração do SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
