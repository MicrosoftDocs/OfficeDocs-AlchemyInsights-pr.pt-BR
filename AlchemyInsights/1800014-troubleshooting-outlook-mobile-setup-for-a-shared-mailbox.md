# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Solucionando problemas de instalação móvel do Outlook para uma caixa de correio compartilhada

Você não pode acessar a caixa de correio compartilhada do app Outlook que está instalado no dispositivo móvel. Você deseja que esse recurso? Informe a equipe de engenharia do Outlook. [Votar aqui](https://go.microsoft.com/fwlink/?linked=862116)!
  
Você pode apenas usar uma caixa de correio compartilhada em um dispositivo móvel, abra seu navegador, entrando no Office 365 e vá Outlook na web. Para obter instruções, consulte [Adicionar uma caixa de correio compartilhada no Outlook na web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Você poderá acessar a caixa de correio compartilhada do Outlook na web.
