# <a name="use-your-browser-to-check-spelling"></a>Use o navegador para verificar ortografia

Não há nenhuma opção de verificação ortográfica no Outlook.com. Para verificar a ortografia, você precisará usar seu navegador da web. Verificação ortográfica está disponível no Microsoft Edge, Internet Explorer 10 e versões posteriores e as versões atuais do Firefox, Chrome e Safari. Verifique as opções para o seu navegador da web saber mais sobre como verificar a ortografia.
