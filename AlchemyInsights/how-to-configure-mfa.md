# <a name="configure-multi-factor-authentication"></a>Configurar a autenticação multifator

Para configurar a MFA, confira [Configurar a autenticação multifator](https://docs.microsoft.com/office365/admin/security-and-compliance/set-up-multi-factor-authentication).

Observe que você precisa ser um administrador global para habilitar e configurar a MFA.
