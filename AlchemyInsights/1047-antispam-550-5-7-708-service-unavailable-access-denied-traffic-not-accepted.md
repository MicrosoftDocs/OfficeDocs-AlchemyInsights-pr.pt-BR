# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 acesso negado, não aceita tráfego desse IP

Este erro ocorre quando o envio de email de reputação conhecida, baixa endereços IP que são normalmente usados por novos clientes.
  
Se você deve enviar email a partir esses endereços IP antes de você pode adquirir licenças, contate o suporte para solicitar uma exceção até que você está possível adquirir licenças.
