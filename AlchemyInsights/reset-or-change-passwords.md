# <a name="reset-or-change-passwords"></a>Redefinir ou alterar senhas

Um usuário pode alterar sua própria senha de **Minha conta** Segurança  >  [& privacidade](https://portal.office.com/account/#security).
  
Se o usuário tiver esquecido sua senha, um administrador poderá redefinir a senha indo para **Usuários**  >  [Usuários ativos,](https://portal.office.com/adminportal/home#/users)selecione os usuários e clique em **Redefinir senha**.
  
[Clique aqui para saber mais](https://docs.microsoft.com/microsoft-365/admin/add-users/reset-passwords), ou para permitir que os usuários redefinir suas próprias senhas esquecidas.
