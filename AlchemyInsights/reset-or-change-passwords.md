# <a name="reset-or-change-passwords"></a>Redefinir ou alterar senhas

Um usuário pode alterar sua própria senha de [minha conta, privacidade &amp; de segurança](https://portal.office.com/account/#security).
  
Se o usuário esqueceu a senha, um administrador pode redefinir a senha indo para usuários [ \> ativos](https://portal.office.com/adminportal/home#/users)do usuário, selecione o (s) usuário (s) e clique em **Redefinir senha**.
  
[Clique aqui para saber mais](https://support.office.com/article/admins-reset-office-365-business-passwords-7a5d073b-7fae-4aa5-8f96-9ecd041aba9c)ou para permitir que os usuários redefinam suas próprias senhas esquecidas.
