# <a name="reset-or-change-passwords"></a>Redefinir ou alterar senhas

Um usuário pode alterar sua própria senha de segurança de **minha conta**  >  [& privacidade](https://portal.office.com/account/#security).
  
Se o usuário esqueceu a senha, um administrador pode redefinir a senha **indo para**  >  [usuários ativos](https://portal.office.com/adminportal/home#/users)do usuário, selecione o (s) usuário (s) e clique em **Redefinir senha**.
  
[Clique aqui para saber mais](https://docs.microsoft.com/microsoft-365/admin/add-users/reset-passwords)ou para permitir que os usuários redefinam suas próprias senhas esquecidas.
