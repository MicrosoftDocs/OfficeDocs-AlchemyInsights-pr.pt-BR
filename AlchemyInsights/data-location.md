# <a name="data-location"></a>Local dos dados

Você pode exibir a localização do seu locatário no centro de administração navegando até a [página de **perfil da organização** ](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) e selecionando local dos **dados**.

Para aplicativos nos quais você não está inscrito, confira [onde seus dados do cliente Microsoft 365 estão armazenados](https://docs.microsoft.com/office365/enterprise/o365-data-locations).
