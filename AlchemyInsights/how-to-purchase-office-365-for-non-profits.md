# <a name="microsoft-365-for-nonprofits---under-review"></a>Microsoft 365 para não fins lucrativos-sob revisão

Sua elegibilidade como organização sem fins lucrativos qualificados da Microsoft está **em revisão**. Você pode esperar receber uma notificação sobre a determinação de suas horas de elegibilidade **48** no momento do envio. Aguarde pelo menos **48 horas** para receber esta notificação antes de abrir uma solicitação de serviço. 

Para obter mais informações, consulte [o Microsoft 365 para não fins lucrativos](https://www.microsoft.com/nonprofits/microsoft-365).
