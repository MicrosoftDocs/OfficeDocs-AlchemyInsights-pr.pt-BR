# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Usar condições ou fórmulas avançadas em um fluxo

Você pode personalizar um fluxo para que ele execute tarefas somente se uma condição for verdadeira. Para obter informações, consulte [Adicionar uma condição a um fluxo](https://go.microsoft.com/fwlink/?linkid=872112).
  
Para criar uma fórmula avançada, confira [esquema de idioma de definição de fluxo de trabalho para aplicativos de lógica do Azure](https://aka.ms/logicexpressions).
