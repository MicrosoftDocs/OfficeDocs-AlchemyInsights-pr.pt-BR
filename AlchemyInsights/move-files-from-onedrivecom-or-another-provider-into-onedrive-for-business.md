# <a name="move-files-from-onedrivecom-or-another-provider-into-onedrive-for-business"></a>Mover arquivos do OneDrive.com ou de outro provedor para o OneDrive for Business

Se você tiver acesso local aos seus arquivos, você pode migrar os arquivos para o OneDrive for Business carregando-os em um navegador da web ou usando o cliente de sincronização do OneDrive. Temos ferramentas adicionais para ajudá-lo com a migração:

- Obtenha uma visão geral abrangente das maneiras que você pode mover seu conteúdo para o OneDrive for Business. Para mais informações, confira [Migrar seu conteúdo para o Microsoft 365](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).
- Você vai precisar de usar diferentes etapas para mover seus blocos de anotações do OneNote. Para mais informações, confira [Mover um bloco de anotações do OneNote para o OneDrive](https://support.office.com/article/move-a-onenote-notebook-to-onedrive-0af0a141-0bdf-49ab-9e50-45dbcca44082).
    
- Configurar Arquivos Sob Demanda. Você pode usar Arquivos Sob Demanda tanto para Windows como para Mac. Para mais informações, confira [Economizar espaço em disco com Arquivos do OneDrive Sob Demanda para Windows 10](https://support.office.com/article/Save-disk-space-with-OneDrive-Files-On-Demand-for-Windows-10-0e6860d3-d9f3-4971-b321-7092438fb38e) e [Economizar espaço em disco com Arquivos Sob Demanda para Mac](https://support.office.com/article/Save-disk-space-with-OneDrive-Files-On-Demand-for-Mac-529f6d53-e572-4922-a585-e7a318c135f0).
- Faça backup das pastas importantes (pastas da Área de trabalho, Documentos e Imagens) em seu computador Windows com o backup de pastas do computador do OneDrive, para que elas fiquem protegidas e disponíveis em outros dispositivos. Para mais informações, confira [Faça backup de suas pastas de Documentos, Imagens e Área de Trabalho com o OneDrive](https://support.office.com/article/back-up-your-documents-pictures-and-desktop-folders-with-onedrive-d61a7930-a6fb-4b95-b28a-6552e77c3057) e [Como o OneDrive protege seus dados na nuvem](https://support.office.com/article/how-onedrive-safeguards-your-data-in-the-cloud-23c6ea94-3608-48d7-8bf0-80e142edd1e1). 
- Quando você move arquivos compartilhados para o OneDrive for Business, eles são reconhecidos como novos arquivos, então você pode querer compartilhar seus arquivos novamente. Para mais informações, confira [Compartilhar arquivos no OneDrive for Business](https://support.office.com/article/Move-files-from-OneDrive-to-OneDrive-for-Business-7fb28cad-7e25-451f-8b4b-2d1a71e5c0e9#sharefiles).
