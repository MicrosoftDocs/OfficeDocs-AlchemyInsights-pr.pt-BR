# <a name="disable-access-to-yammer"></a>Desabilitar o acesso ao Yammer

A solução para desabilitar o acesso ao Yammer é bloquear e cancelar a atribuição de licenças do Yammer para os usuários do Office 365. Confira o artigo a seguir para obter mais informações.
  
- [Desativar o acesso do Yammer para usuários do Office 365](https://support.office.com/article/1f79bfad-f713-4143-aa5d-5584985ce53a)
