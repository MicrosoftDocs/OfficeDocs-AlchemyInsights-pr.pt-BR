# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Mapear uma biblioteca do SharePoint para uma unidade de rede

Mapeamento de uma biblioteca como uma unidade de rede é temporário e com suporte apenas através do Internet Explorer. Em vez disso, [sincronizar arquivos do SharePoint com o novo cliente de sincronização do OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) que fornece [Arquivos sob demanda](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Acesse todos os seus arquivos no OneDrive sem usar o espaço de armazenamento local.
  
[Obtenha ajuda Configurando e Solucionando problemas de unidades de rede mapeadas](https://go.microsoft.com/fwlink/?linkid=872946)
