# <a name="repeated-login-to-open-vtirpc"></a>Logon repetida para abrir _vti_rpc

Ao trabalhar em arquivos de OneDrive usando o Office 2016, você poderá ver a mensagem: "Conte-na conta que você deseja usar para abrir _vti_rpc." Esse problema foi corrigido na versão 1708 (16.0.8431.2094). Para corrigir esse problema, atualize para a versão mais recente do Office. Selecione **arquivo** \> **conta** \> **Atualizar opções** \> **Atualizar agora**.
