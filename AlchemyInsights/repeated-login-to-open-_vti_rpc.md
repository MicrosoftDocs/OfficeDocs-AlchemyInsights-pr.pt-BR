# <a name="repeated-login-to-open-_vti_rpc"></a>Login repetido para abrir _vti_rpc

Ao trabalhar com arquivos do OneDrive usando o Office 2016, você poderá ver a mensagem: "Diga a conta que você deseja usar para abrir o _vti_rpc". Esse problema foi corrigido na versão 1708 (16.0.8431.2094). Para corrigir isso, atualize para a versão mais recente do Office. Selecione **** \> **** \> atualizar opções \> de **atualização** de conta de arquivo **agora**.
