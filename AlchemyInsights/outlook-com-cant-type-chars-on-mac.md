# <a name="how-to-type-special-characters-on-a-mac"></a>Como digitar caracteres especiais em um Mac

Para digitar caracteres especiais no Mac, abra o menu de **caracteres especiais** com o atalho de teclado ⌘ de controle de **comando**  +  **Control**  +  **Space**.

Um atalho rápido para @ também é **ALT**  +  **64**.
