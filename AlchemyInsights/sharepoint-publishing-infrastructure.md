# <a name="sharepoint-publishing-infrastructure"></a>Infraestrutura de publicação do SharePoint


<p>O SharePoint inclui um conjunto clássico de recursos que permitem que você crie e publique páginas da Web sofisticadas para sua intranet. Ative a infraestrutura de publicação do SharePoint no nível do conjunto de sites para ativá-las.</p>  <p>Para obter mais informações, visite os links abaixo.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Habilitar recursos de publicação</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Recursos de publicação</a></li>  </ul>  <p>Observações:</p>  <ul>  <li>Os modelos de site de publicação clássicos incluem os recursos de publicação por padrão.</li>  <li>Os recursos de publicação clássica não estão disponíveis em sites modernos. Os sites modernos fornecem uma alternativa mais recente por padrão.</li>  </ul>
