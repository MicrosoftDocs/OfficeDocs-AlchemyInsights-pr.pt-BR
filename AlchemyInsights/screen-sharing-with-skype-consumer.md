# <a name="screen-sharing-with-skype-consumer-users"></a>Compartilhamento de tela com usuários de consumidor do Skype

Para obter instruções gerais sobre como compartilhar seu programa, para saber mais sobre restrições com compartilhamento e para solucionar problemas de recursos, revise Compartilhar sua tela [no Skype for Business](https://support.microsoft.com/office/share-and-present-content-from-skype-meetings-app-skype-for-business-web-app-234b0c06-a88d-4707-904c-4fd6c571fc01).  

Atualmente, os usuários do Skype for Business não podem compartilhar a tela com os usuários de consumidor do Skype. Clique [aqui para fornecer](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) comentários sobre a habilitação do compartilhamento de tela com usuários do Skype do consumidor.
