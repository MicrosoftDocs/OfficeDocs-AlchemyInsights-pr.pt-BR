# <a name="screen-sharing-with-skype-consumer-users"></a>Compartilhamento de tela com usuários de consumidor do Skype

Para obter instruções gerais sobre como compartilhar seu programa, para saber mais sobre restrições com o compartilhamento e para solucionar problemas de recursos, revisar [compartilhar sua tela no Skype for Business](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

Atualmente, os usuários do Skype for Business não conseguem compartilhar a tela com usuários de consumidor do Skype. [Clique aqui](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) para fornecer comentários sobre como habilitar o compartilhamento de tela com usuários do Skype do consumidor.
