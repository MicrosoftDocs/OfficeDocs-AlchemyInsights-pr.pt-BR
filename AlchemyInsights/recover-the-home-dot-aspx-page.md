# <a name="recover-the-homeaspx-page"></a>Recupere a página Home.aspx

Se a página inicial do site for excluída, ela poderá ser recuperada da Lixeira acessando a URL da Lixeira.

1. Navegue até https://\<tenantname>.sharepoint.com/sites/\<sitename>/_layouts/15/RecycleBin.aspx?, substituindo <**tenantname**> com o nome do locatário e <**sitename**> com o nome do seu site.

2. Select the page to restore.

Se a página inicial ainda não puder ser encontrada, você poderá criar uma nova página do site e [configurá-la como sua página inicial](https://support.microsoft.com/en-gb/office/use-a-different-page-for-your-sharepoint-site-home-page-35a5022c-f84a-455d-985e-c691ab5dfa17?ui=en-us&rs=en-gb&ad=gb). O erro 404 não encontrado não deve mais ser exibido quando você tentar acessar o site.
