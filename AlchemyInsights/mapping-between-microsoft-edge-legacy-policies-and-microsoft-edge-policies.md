# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Saiba mais sobre o mapeamento entre as políticas do Versão Prévia do Microsoft Edge e as políticas do Microsoft Edge

As políticas de grupo do Versão Prévia do Microsoft Edge (versões 45 e anteriores) e as configurações do Mobile Device Management (MDM) são mapeadas para as políticas relevantes do Microsoft Edge, versão 80. Para obter mais informações, consulte [Mapeamento de política de Versão Prévia do Microsoft Edge e Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
