# <a name="how-to-create-a-file-request"></a>Como criar uma solicitação de arquivo

O recurso de solicitação de arquivo no OneDrive permite escolher uma pasta onde outras pessoas podem carregar arquivos usando um link que você os envia. Os usuários dos quais você solicita arquivos não podem ver o que está na pasta-- eles só podem carregar arquivos para ela.

**Observação**: um administrador deve habilitar [links de](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) Qualquer Pessoa no OneDrive para que as solicitações de arquivo funcionem.

- Para obter mais informações, consulte [Create a file request](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af).
