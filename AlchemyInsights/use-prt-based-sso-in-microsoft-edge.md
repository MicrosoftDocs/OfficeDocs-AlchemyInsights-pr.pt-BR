# <a name="use-prt-based-sso-in-microsoft-edge"></a>Usar o SSO baseado em PRT no Microsoft Edge

Um [Token de Atualização Principal (PRT)](https://go.microsoft.com/fwlink/?linkid=2133632) é uma chave do Azure Active Directory usada para autenticação em dispositivos Windows 10, iOS e Android. A PRT habilita o SSO para todos os aplicativos usados nesses dispositivos.

No Microsoft Edge, o suporte para SSO baseado em PRT é nativo e não requer extensões adicionais. No Windows 10 RS3 e acima, os usuários assinados obterão o SSO com a PRT para sites que suportam o SSO baseado em PRT.
