# <a name="app-configuration-policies-for-microsoft-intune"></a>Políticas de Configuração do Aplicativo para o Microsoft Intune

As políticas de configuração do aplicativo no Microsoft Intune fornecem configurações para aplicativos iOS ou Android. Essas configurações permitem a personalização de um aplicativo. O suporte à política de configuração é limitado por versões do sistema operacional e aplicativos.

Para obter mais detalhes, confira [Políticas de Configuração do Aplicativo para o Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
