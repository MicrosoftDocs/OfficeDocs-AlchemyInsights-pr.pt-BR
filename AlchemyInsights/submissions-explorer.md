# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Novo recurso para ajudar a diagnosticar EOP e ATP

Como administrador, agora você pode facilmente relatar mensagens de email suspeitas, URLs e conteúdo para a Microsoft no [Gerenciador de envios](https://protection.office.com/reportsubmission). Você pode até mesmo usá-lo para ajudar a entender por que uma mensagem legítima foi sinalizada incorretamente. Para obter mais informações, consulte este [comunicado](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
