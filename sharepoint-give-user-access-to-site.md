# <a name="give-users-access-to-sharepoint-site"></a>Conceder aos usuários acesso ao site do SharePoint

<p>O recurso de solicitação de acesso permite que as pessoas solicitem acesso ao conteúdo que elas não têm permissão para ver atualmente. Como proprietário do site, você pode configurar o recurso para que ele envie um email quando alguém solicitar acesso a um site. Para obter mais informações, visite <a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3">configurar e gerenciar solicitações de acesso</a>.</p>  <p>Aqui estão alguns links comuns referentes a solicitações de acesso:</p>  <ul>  <li><a href="https://support.office.com/en-us/article/-Access-Denied-to-Access-Requests-list-or-Request-approval-failed-when-you-process-a-pending-request-in-SharePoint-Online-3ec71fe1-9111-4a4c-84a5-bdfe05c2e558">"Acesso negado" para acessar a lista de solicitações ou "solicitar falha de aprovação" quando você processa uma solicitação pendente no SharePoint Online</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow">Alterar as configurações de solicitações de acesso</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462557">Aprovar ou recusar solicitações de acesso para um site</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558">Gerenciar convites para usuários externos (somente SharePoint Online)</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260">Retirar convites</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560">Reenviar um convite</a></li>  </ul>
