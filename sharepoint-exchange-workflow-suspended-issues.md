# <a name="workflows-in-sharepoint"></a>Workflows in SharePoint

<p>Se os fluxos de trabalho do SharePoint não estiverem enviando emails, sua organização pode ter encontrado os limites de remetentes do Exchange Online.&nbsp;</p> <p>Mensagem de erro "o fluxo de trabalho é suspenso" pode ocorrer se você tiver um dos seguintes itens:</p> <ul> <li>Você tem um fluxo de trabalho no SharePoint Online que está usando o tipo de plataforma de fluxo de trabalho do SharePoint 2010 ou SharePoint 2013.</li> <li>O fluxo de trabalho é configurado para enviar uma mensagem de email personalizada para mais de 200 usuários por vez, mais de 10.000 destinatários por dia ou mais de 30 mensagens por minuto.</li> <li>Quando você executa o fluxo de trabalho, a mensagem de email não é enviada e você observa a mensagem de erro, o <strong>status interno é definido como suspenso</strong> ou <strong>não é possível enviar a um destinatário</strong> .</li> </ul> <p>Para obter mais informações, consulte o <a href="https://support.office.com/en-us/article/-daily-email-limit-has-exceeded-and-your-workflow-has-been-suspended-or-unable-to-send-to-a-recipient-error-in-a-sharepoint-online-workflow-89d02169-5fa6-4259-affc-73edb6ca9fb6?ui=en-US&amp;rs=en-US&amp;ad=US">artigo</a>a seguir.</p>
