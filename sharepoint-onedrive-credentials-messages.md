# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Mensagens de credenciais no SharePoint ou no OneDrive

<p>Os usuários podem ser solicitados periodicamente pelo Office para credenciais para o SharePoint Online ou o OneDrive ao tentar abrir documentos.</p> <p>Veja a seguir alguns exemplos de mensagens de credenciais:</p> <p>&bull;Entrar com sua conta organizacional <span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span></p> <p>&bull;As credenciais são necessárias <span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span></p> <p>&bull;Não é possível conectar agora, verifique sua conexão de rede. <span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span></p> <p>Se você estiver enfrentando esses sintomas, tente as etapas descritas neste <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">artigo</a>.</p> <p>Como alternativa, se você estiver usando o logon único (SSO) para autenticar, consulte <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">Este artigo</a> em vez disso.</p> <p>&nbsp;</p>
